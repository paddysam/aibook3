# 第 05 章细纲 由我带路

本章定位：
第五章要完成前五章节奏里的最后一个关键目标：`陆沉正式掌握带队资格`。第四章已经让所有人看到“不听陆沉的话就会出事”，这一章就要把这种被动服气，变成主动跟随。与此同时，本章还要埋下下一阶段的核心危机：`镜子后面的人`。

本章目标：
- 让陆沉第一次以“带路人”的身份组织幸存者行动
- 让周岚等现实阻力角色开始被迫服从
- 让林晚明确站到陆沉这一边
- 让 `救人会让规则视界变清晰` 的成长逻辑再往前推进一步
- 在章尾抛出主角自己的死法，为第6章镜中替身做钩子

本章核心情绪：
从第四章末尾众人把希望压到陆沉身上的窒息感，推进到“主角被迫接管局面”的紧张，再在短暂建立秩序后，突然把危险转回到陆沉自己身上。

承接开头：
- 紧接第四章结尾
- 韩峥当众翻车后，十三层现场已经接近崩盘
- 所有人第一次把“怎么活”这个问题抛给了陆沉

场景 1：陆沉被迫接管局面

内容要点：
- 韩峥出事后，幸存者群体彻底乱了，有人尖叫，有人往回退，有人想冲出十三层。
- 这时候最重要的不是讲道理，而是让陆沉先用一句短促、带命令感的话把人按住。
- 比如：
  - 不准乱跑
  - 全都低头
  - 谁再乱数东西，谁先死
- 陆沉自己心里也慌，但他必须在这一刻撑住。
- 这会是主角第一次不是“提醒”，而是“发号施令”。

这一场的作用：
- 完成主角身份上的第一次切换
- 让读者看到他开始真正进入带队状态

场景 2：第一次按规则排人

内容要点：
- 陆沉一边看规则，一边快速决定幸存者的站位和行动顺序。
- 他可以直接下几个非常具体的命令：
  - 谁负责扶着快崩溃的人
  - 谁不准抬头看走廊尽头
  - 谁站最里面，谁靠墙走
  - 所有人经过病房门口都不要停
- 这些命令必须很具体，不能空喊“大家小心”，这样才能体现陆沉的带队能力。
- 众人开始照做，不是因为完全信服，而是因为刚才韩峥的下场太惨，谁都不敢赌。

这一场的作用：
- 让“主角掌握带队资格”落到实操层面
- 让规则求生和人物调度结合起来

场景 3：周岚第一次被陆沉当场喝住

内容要点：
- 周岚毕竟之前一直把陆沉当跑腿的，不可能一下子彻底服气。
- 她可能会因为职业习惯，想去扶韩峥、想去开某扇门、想回头叫人，或者忍不住质疑陆沉的安排。
- 就在她准备踩中某条规则前，陆沉必须当场把她喝住。
- 这里不能太软，最好带一点压迫感，比如：
  - `别动！`
  - `低头！`
  - `你想死别连累别人！`
- 周岚会先愣住，再因为身边突然出现的异常反应，意识到陆沉不是在凶她，而是在救她。
- 她嘴上可能不服，但动作上已经会老老实实照做。

这一场的作用：
- 给主角一波小打脸
- 让原本压主角的人第一次在行动上服从主角
- 为后续“周岚逐步转变”打基础

场景 4：林晚站到陆沉这边

内容要点：
- 和周岚形成对照，林晚会成为第一个主动配合陆沉的人。
- 她不一定会说很多话，但她会帮陆沉维持秩序，比如：
  - 主动让其他人低头
  - 扶住快崩溃的人
  - 告诉大家先听陆沉的
- 她这样做的意义很大，因为这会让陆沉不再像一个人在顶着所有人。
- 两人的关系不用立刻写得很亲密，但要明确建立“她开始成为主角的第一辅助”。

这一场的作用：
- 给主角补上一个稳定合作对象
- 降低纯独角戏的疲劳感

场景 5：十三层里的第一次有序前进

内容要点：
- 陆沉带着众人开始在十三层里移动，这时要写出一种“每走一步都在踩规则边缘”的感觉。
- 可以依次触到三条规则的压迫感，但本章先不让红鞋护士正式登场：
  - 经过病房门时，门内似乎有婴儿哭声传来，但没人敢去找来源
  - 某个转角闪过一抹红色鞋尖，但看不真切
  - 门牌数量似乎会变，谁也不敢再数
- 陆沉会越来越清楚地感觉到：十三层不是死寂的，它一直在观察他们。
- 这里可以安排一个小险情，但不要抢走本章重点，比如有人差点被哭声带偏，被林晚或陆沉及时拽回来。

这一场的作用：
- 维持十三层副本压迫感
- 让读者感到“带队不等于安全，只是死得没那么快”
- 提前预热红鞋护士和婴儿哭声

场景 6：能力再进一层

内容要点：
- 在连续救险、正确带路后，陆沉眼前的规则视界再次发生细微变化。
- 他会发现这次提升不只是规则字变清楚，而是自己开始能看到：
  - 某些地点的危险程度不同
  - 某些人头顶的死亡倒计时更稳定
  - 某些怪谈目标和自己之间有更明确的关联
- 这种升级还是不要写得太网游，更像“感知权限被打开了一层”。
- 陆沉也会因此确认一件事：
  - 他不只是活下来才会变强
  - 他在带人活下来这件事上，也能变强
- 这会进一步推动他从“我先保自己”转向“我得把这一队人尽量带出去”。

这一场的作用：
- 强化成长逻辑
- 让“带队”本身也成为主角的爽点来源

场景 7：护士站镜子异常

内容要点：
- 众人短暂停下时，前方出现一个十三层的废弃护士站。
- 环境可以写得很诡：台面有灰，却像刚被人碰过；玻璃柜里东西摆得整整齐齐；镜面有水汽一样的雾痕。
- 陆沉刚想带人绕开，规则视界却在护士站镜子附近疯狂闪烁。
- 这时他第一次从自己身上看到完整一点的危险提示：
  - `剩余存活时间：00:10:00`
  - `死亡触发：镜子后面的人`
- 注意，这里不要立刻解释“镜子后面的人”是什么，只要把这句死法扔出来。
- 这会让整章最后一下从“主角开始带人”瞬间翻到“下一个最危险的人就是主角自己”。

这一场的作用：
- 给第6章镜中替身立核心钩子
- 把压力从群体危机收束到主角个人危机
- 让读者继续追下一章

章尾钩子：
建议收在一句非常干脆的死法提示上，比如：
`陆沉第一次看见自己的死法：十分钟后，他会死在镜子后面的人手里。`

本章节奏提醒：
- 前半章重点不是炫设定，而是写“主角怎么把一群快崩的人稳住”
- 中段要让带队的过程有具体动作，不要只写心理变化
- 后半章必须把危险重新收回到主角自己身上，形成新的追读钩子

本章伏笔：
- 红鞋护士还没正式出现，但已经在十三层里留下存在感
- 婴儿哭声对应的东西后面一定会成为大威胁
- 护士站镜子和十三层的病房、门牌规则可能属于同一层更深的怪谈结构

本章读者爽点：
- 主角第一次真正带队，别人开始按他的命令活命
- 原本压他的角色开始被他当场喝住
- 主角能力继续升级，且升级和“带人活下去”直接相关
