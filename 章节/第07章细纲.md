# 第 07 章细纲 病人就是我

本章定位：
第七章要把第六章留下的 `门牌背后全写陆沉` 推成一次真正的身份压迫。前面五章和第六章更多是“外部规则杀人”，这一章要把恐惧转向“怪谈正在改写主角身份”，同时让赵虎不再只是嘴硬，而是开始真正煽动队伍分裂。

本章目标：
- 放大门牌背后全写陆沉的压迫感
- 让怪谈第一次用“你就是病人”的方式攻击主角心态
- 让赵虎借机煽动部分人质疑陆沉
- 让陆沉短暂动摇，但不能崩
- 让陆沉决定反过来利用赵虎
- 章尾把红鞋护士正式抬上台前

本章核心冲突：
- 主冲突：陆沉开始怀疑自己和十三层是不是有更深关系
- 次冲突：赵虎趁机煽动幸存者内部裂开
- 恐怖来源：身份错位、门牌暗示、镜中自我、被怪谈“定义”

本章回报类型：
- 不是避死也不是救人
- 本章回报重点是：`主角稳住自我，没有被怪谈带偏，并主动决定设局`

承接开头：
- 紧接第六章章尾
- 林晚已经发现所有门牌背后都写着陆沉
- 队伍还停在十三层深处，镜像怪谈的余压仍在

场景 1：所有门牌背后都写着陆沉

内容要点：
- 林晚和陆沉再确认几扇门牌背后，结果都一样。
- 其他幸存者也逐渐意识到不对，恐慌会从“外面有鬼”变成“陆沉是不是本来就和这里有关”。
- 不同人的反应要区分：
  - 林晚第一反应是担心陆沉被怪谈针对
  - 周岚本能害怕，但还没敢说出口
  - 某些家属会开始怀疑是不是陆沉把他们带到这里
  - 赵虎则会敏锐地抓住这个机会

这一场的作用：
- 把上章钩子变成真正的剧情压力
- 让内部裂痕有现实基础

场景 2：镜子和门牌共同施压

内容要点：
- 众人移动时，十三层开始出现一连串更明显的“针对陆沉”的异常：
  - 镜面里某个门牌号和现实对应不上
  - 某扇病房门口像是刚有人写下“陆沉”两个字
  - 远处病房里传来一句模糊的人声，像是在喊“病人陆沉”
- 规则视界这时不给明确答案，只给零碎的压迫信息。
- 陆沉会第一次真正产生短暂怀疑：自己是不是从一开始就被选中了？甚至是不是曾经来过这里？

这一场的作用：
- 把身份压迫感从“线索”升级成“连续攻击”
- 让读者开始跟着怀疑，但又没有完整答案

场景 3：赵虎趁机煽动

内容要点：
- 赵虎这章不能只会阴阳怪气，要让他开始像一个真正危险的人。
- 他会借门牌背后的名字煽动大家：
  - 这里只有陆沉的名字
  - 陆沉总能先一步看到怪事
  - 会不会这一切本来就和他有关
- 他说这些话时，不是完全相信，只是想把压力推回陆沉身上，同时抢回一点主导权。
- 队伍里会有至少一两个人被他说动，开始和陆沉拉开距离。

这一场的作用：
- 让赵虎真正从“刺头”升级成“会坏事的人”
- 给第8章陆沉反杀他更强的合理性

场景 4：主角短暂动摇

内容要点：
- 陆沉不是神，这里必须给他一个短暂的失衡。
- 可以用一个更私人的刺点：
  - 他想起自己关于医院的记忆过于空白
  - 他意识到自己竟然一点都不知道十三层为什么会盯上自己
  - 镜中那个慢半拍的自己像是在默认“你本来就属于这里”
- 这个动摇不能太长，但必须真实。
- 林晚这时最关键的作用不是安慰，而是把他从“被怪谈定义”里拽回来。
- 她可以给一句很实用的话：
  - 不管这里为什么写你的名字
  - 至少到现在，只有你一直在救人

这一场的作用：
- 让主角更像人，而不是一直稳定输出
- 林晚和主角的关系再往前推进一点，但不写成感情戏

场景 5：陆沉反过来利用赵虎

内容要点：
- 陆沉被林晚一句话拉回来后，思路开始逆转。
- 他意识到：怪谈想让自己怀疑自己，赵虎也在配合怪谈分裂队伍。
- 既然赵虎这么急着抢主导权，那就反过来把最危险的东西引向他。
- 这时规则视界能给到一点点新信息：
  - 赵虎头顶开始出现模糊的死亡触发
  - 虽然还看不完整，但明显和“抬头、回头、出声”之类行为有关
- 陆沉不会现在就动手，而是在心里正式决定：赵虎可以拿来当诱饵。

这一场的作用：
- 把主角从被动承压转回主动设局
- 为第8章规则反杀打逻辑基础

场景 6：红鞋护士的前兆

内容要点：
- 这一章不能让红鞋护士直接全面登场，要先做压迫前兆。
- 最好从“听觉”切进来，和前面镜像、门牌拉开：
  - 走廊尽头传来规律的高跟鞋声
  - 一开始很远，后来越来越近
  - 每响一下，灯光都像暗一点
- 众人刚被赵虎煽乱过，现在再听到这个声音，队伍会瞬间紧绷。
- 陆沉马上想起十三层规则：
  - 如果看见穿红鞋的护士，必须先低头，再后退
- 这条规则之前一直没被真正验证，这里就是它第一次“活过来”。

这一场的作用：
- 把第8章主危机拉出前奏
- 让读者带着强预期进下一章

章尾钩子：
- 适合收在一个短促但画面很强的声音点上，比如：
`走廊尽头，高跟鞋的声音停了。下一秒，一双鲜红的鞋尖，缓缓停在了灯影边缘。`

本章节奏提醒：
- 身份压迫要真，但不能写成长篇自怨自艾
- 赵虎一定要升级为“能带偏队伍的人”
- 林晚这章的作用是把陆沉拉回现实，而不是只负责害怕
- 章尾必须把危险切到红鞋护士，形成下章硬钩子

本章伏笔：
- 陆沉和医院、十三层之间可能确实有更深关联
- 赵虎的死亡触发已经埋下
- 红鞋护士的规则将在下一章正式被用来反制人和怪谈

本章读者爽点：
- 主角没有被怪谈的“身份定义”击垮
- 主角第一次决定主动设局，而不是被动求生
- 红鞋护士出场前兆足够拉追读
