# 第 01 章细纲 血月值夜

本章定位：
开篇立刻建立 `低身份主角 + 怪异预警 + 血月降临 + 第一次死人`，并且在第一章内完成一个清晰闭环：陆沉先看到死亡提示，再眼睁睁看着提示应验，让读者立刻知道这不是普通都市文，而是带强规则钩子的都市怪谈爽文。

本章目标：
- 让读者记住陆沉的身份处境，形成天然代入和期待打脸
- 提前埋入“不能看月亮”“它会记住你”这条关键怪谈预警
- 用医院场景制造天然压迫感和封闭感
- 让陆沉第一次“想救人却救不下来”，把无力感和钩子一起立住
- 在章尾给出明确死人画面，完成第一强钩子

本章出场人物：
- 陆沉：精神病院夜班护工，谨慎、寡言、长期被轻视
- 护士长周岚：嘴硬势利，把脏活累活都甩给陆沉
- 病人家属：情绪激动，把医院和陆沉当出气筒
- 预警病人：神神叨叨，反复强调今晚不能看月亮
- 值班护士许薇：章尾暴毙对象，用来完成第一轮恐惧验证

开篇气氛：
不要直接解释世界观，先写陆沉值夜时的压抑日常。
重点突出：
- 深夜医院本身就阴冷
- 陆沉在这里没有话语权
- 所有人都把他当最底层的人使唤
- 这部分一定要短，不要把现实压抑写成完整职场戏，尽快切进怪异

场景 1：值夜开局，先压主角

内容要点：
- 从凌晨前的精神病院写起，走廊灯光惨白，空气里有消毒水和陈旧潮气。
- 陆沉正在做最脏最累的夜班杂活，比如收拾病房、处理闹事病人留下的狼藉、替护士跑腿。
- 护士长周岚语气很冲地使唤他，顺手把责任也往他头上扣，体现他地位低。
- 病人家属因为探视受阻和医院争吵，最后把怒气发到陆沉身上，陆沉只能忍。

这一场的作用：
- 第一时间告诉读者，主角起点低，后面打脸会更爽
- 让读者先对陆沉产生一点憋屈感
- 但篇幅不要过长，最好在开章较短篇幅内结束，尽快切入怪异预警

场景 2：怪人预警，埋下第一条规则

内容要点：
- 陆沉去送药或巡视时，遇到一个情绪很不稳定的老病人。
- 老病人表现得像在胡言乱语，但句子要短、重复、带画面感。
- 老病人反复说：
  - 今晚不能看月亮
  - 看见月亮的人会被记住
  - 一旦被记住，就回不去了
- 陆沉起初只当对方病情发作，但仍然下意识记住这几句话。
- 这时候可以让老病人突然抓住陆沉手腕，眼神异常清醒一瞬，制造反差。

这一场的作用：
- 抛出第一条规则雏形
- 让读者知道这些疯话后面一定会应验

场景 3：异常逐渐出现，日常开始失真

内容要点：
- 医院里陆续出现几处说不清的异常，但都先控制在“似真似幻”的程度。
- 比如：
  - 某个病房呼叫铃无故响起，但进去后没人
  - 监控画面里有一段走廊比现实更长
  - 电梯数字短暂跳出不存在的楼层
- 陆沉因为长期值夜，对这些细节更敏感，隐约察觉今晚不对劲。
- 其他人却忙着工作和争吵，没人把这些异常当回事。

这一场的作用：
- 提前把怪谈氛围做起来
- 不一下子把超自然掀满，给血月出现留空间
- 这里只挑两三个最醒目的异常，不要一口气铺太多怪点

场景 4：血月降临，世界正式翻面

内容要点：
- 接近午夜，整栋住院楼突然断电，备用灯几秒后才亮起，形成短暂黑暗。
- 黑暗里要写出声音细节，比如远处病房拍门声、广播电流杂音、有人惊叫。
- 灯光恢复后，窗外已经出现血色月亮，颜色不自然，像浸在血水里。
- 所有监控画面同时出现雪花，部分屏幕里像闪过了不该存在的人影。
- 医护和家属开始慌，但还在试图用停电、设备故障解释一切。

这一场的作用：
- 把“怪异”升级成“异变”
- 让读者明确故事正式进入主线

场景 5：规则视界初次出现

内容要点：
- 陆沉在混乱中再次看向走廊，视野里第一次浮现陌生文字或半透明提示。
- 文案不要一次讲太多，只给出最关键的一条视觉冲击。
- 比如他看到值班护士许薇头顶浮现：
  - `剩余存活时间：00:01:47`
  - `死亡触发：回头`
- 陆沉先怀疑自己是不是太累出现幻觉，但数字在持续跳动。
- 他想提醒许薇，却因为自己平时没分量，别人根本不信，甚至觉得他晦气。
- 可以安排周岚或其他人顺手把他喝住，让“主角知道危险却没人听”这件事更扎心。

这一场的作用：
- 主角能力第一次亮相
- 制造“知道却来不及阻止”的紧张感
- 让读者立刻明白这个能力最直接的用法，就是看见死法

场景 6：第一人死亡，章尾爆钩

内容要点：
- 走廊深处传来一点异常动静，像有人轻声叫名字，或者有东西擦过墙面。
- 许薇本能回头，动作很自然，正好踩中刚刚显示的死法。
- 下一秒死亡要写得快、狠、突然，不能拖。
- 可以写成她整个人僵住，眼球迅速充血，随后直挺挺倒下，或者脖颈出现诡异扭转。
- 周围人瞬间炸锅，有人尖叫，有人冲过去，有人后退。
- 陆沉看着地上的尸体，再抬头时，视野里又浮现出更多看不懂的规则残字。

章尾钩子：
必须收在一个非常短促的强画面上，比如：
`许薇只是回头看了一眼，下一秒就死了。`

本章节奏提醒：
- 前半章用现实压抑托住人物处境，但占比不要超过全章三分之一
- 后半章逐步加怪，不要上来就全解释
- 第一章不要解释 `规则视界` 从哪来，只负责把“看到提示”和“提示应验”做成闭环
- 章尾一定要死人，而且死因要和前面提示形成闭环

本章伏笔：
- 老病人的“不要看月亮”
- 电梯异常楼层
- 监控里不该出现的人影
- 陆沉为何能看见规则

本章读者爽点：
- 不是传统爽点，而是“钩子爽点”
- 读者会因为主角突然能看到死亡提示而想继续追
