# 第 04 章细纲 封死的十三层

本章定位：
第四章要完成前五章节奏里的第一个关键转折：`十三层正式开启 + 强势角色吃亏 + 陆沉从被动提醒变成被迫带路`。这一章的重点不是单纯“加新怪谈”，而是让所有人第一次正面意识到，陆沉说的话不是危言耸听，而是唯一能救命的东西。

本章目标：
- 让十三层作为第一个正式副本入口登场
- 让医院幸存者开始尝试抱团，但内部仍然不信任陆沉
- 让副院长韩峥作为强势角色亲自吃亏，完成一次明显打脸
- 让陆沉第一次读出十三层的核心规则
- 在章尾把“现在该怎么活”这个选择题抛到陆沉面前

本章核心情绪：
从第三章的“刚救下一个人、众人开始动摇”，迅速推进到“所有退路都失效、必须面对十三层”。整章要有一种路越走越窄的压迫感。

承接开头：
- 紧接第三章结尾
- 电梯在无人操作的情况下停在了 `13`
- 林晚明确说出：住院楼的十三层早就封死，正常情况下根本不存在

场景 1：十三层亮起，所有人先慌再否认

内容要点：
- 电梯数字停在 `13` 后，所有人第一反应不是进去，而是本能后退。
- 林晚、周岚、家属、护工全都被这个数字刺激到，因为这已经超出了“设备故障”的解释范围。
- 但越是这样，越会有人本能地找现实借口来压恐惧。
- 这时副院长韩峥登场最合适。他可以是听到动静赶来的人，也可以本来就在幸存者队伍里。
- 韩峥第一反应是压场，说什么：
  - 医院根本没有十三层
  - 这就是停电导致的线路故障
  - 别被一个护工几句话吓疯
- 他天然站在“身份权威”一边，正好和陆沉形成对冲。

这一场的作用：
- 建立韩峥这个“强势但错误”的角色
- 让大家短暂抱团，但又没人真正有主心骨
- 给后面打脸准备情绪落差

场景 2：所有退路都开始失效

内容要点：
- 有人提议不管十三层，直接下楼逃出去。
- 但医院的空间开始出现明显错位：
  - 楼梯明明往下走，却绕回原楼层
  - 走廊尽头本该是出口，转过去却还是同一段护士站
  - 电梯之外的方向也在一点点变得不对
- 这种“不是你想不去十三层就能不去”的感觉要写出来。
- 陆沉逐渐意识到，十三层不是一个可选项，而是医院异变强行推给他们的下一关。
- 周围人会因此更慌，但仍然没有完全听陆沉的话。

这一场的作用：
- 封死其他退路
- 合理逼出“必须面对十三层”
- 把故事从局部怪谈推进到正式副本

场景 3：十三层入口规则第一次出现

内容要点：
- 电梯门缓缓开出更大缝隙，冷风和血腥味涌出来，里面的灯忽明忽暗。
- 陆沉看向电梯入口，规则视界第一次给出比前面更完整、也更危险的一组规则：
  - `进入十三层后，不要数病房数量`
  - `如果看见穿红鞋的护士，必须先低头，再后退`
  - `听到婴儿哭声时，绝不能寻找声音来源`
- 这些规则要让读者一眼就记住，且和前面广播、307病房形成明显升级感。
- 陆沉这次不是只看到一句死法，而是第一次看到一个“区域规则包”，说明十三层的危险等级更高。
- 但他还没来得及解释完，韩峥就会站出来打断。

这一场的作用：
- 正式建立十三层的规则感
- 让陆沉的能力进一步升级有展示空间
- 给“强势角色不信规则”创造直接冲突

场景 4：韩峥硬压主导权

内容要点：
- 韩峥听完陆沉的话，第一反应不是害怕，而是愤怒和不耐烦。
- 他会觉得一个夜班护工突然发号施令，本身就是在挑战他的权威。
- 他可以当众斥责陆沉：
  - 不要散播谣言
  - 不要用这些疯话吓病人家属
  - 现在听我安排，所有人去监控室或一楼大厅
- 周岚等人会本能偏向韩峥，因为他身份更高、更像“正常世界的秩序”。
- 林晚虽然已经信陆沉一点，但这时也未必敢公开对着副院长说话。
- 这会让陆沉再次陷入一种“明明知道危险却没人愿意完全听”的局面。

这一场的作用：
- 强化现实权力和规则生存的冲突
- 为韩峥后面的翻车积蓄力度

场景 5：正式踏入十三层

内容要点：
- 不管是韩峥强行带路，还是空间错位把大家逼进去，最终所有人都会踏进十三层。
- 一进去就要让氛围跟前面的楼层完全不一样：
  - 病房门牌老旧、数字模糊
  - 墙皮脱落，空气潮湿发冷
  - 明明没有风，门牌却微微晃动
- 最重要的是：这里不像废弃楼层，更像一直有人在“维持运行”。
- 陆沉能感觉到规则视界在这里变得更活跃，说明十三层和前面的走廊规则不是一个级别。
- 人群会本能靠拢，但仍然不安定，韩峥还在试图维持表面秩序。

这一场的作用：
- 完成从医院日常空间到正式副本空间的切换
- 给读者一个明确升级感

场景 6：第一次数病房

内容要点：
- 在十三层里，病房门数量、走廊长度、灯光分布都很反常，人的本能就是想“数清楚”这里到底有多少间房。
- 陆沉看着规则反复闪烁，立刻意识到“不要数病房数量”不是装饰句，而是极高优先级规则。
- 他第一时间警告韩峥和其他人不要数门。
- 韩峥越听越火，觉得这条规则荒谬到可笑，甚至会反问一句：
  - 数个病房也会死人？
- 为了夺回主导权，也为了证明陆沉在胡说，他当众开始数：
  - 一间
  - 两间
  - 三间……
- 前面几声要正常，数到第七间时，异常瞬间爆发。

这一场的作用：
- 把规则从“听起来怪”变成“真的不能碰”
- 给本章最强打脸做蓄力

场景 7：韩峥翻车，现场崩盘

内容要点：
- 数到第七间的瞬间，韩峥的脸上开始出现极其诡异的变化。
- 不一定要直接整张脸消失，可以是更有冲击力的过程：
  - 五官像被无形手掌抹平一半
  - 一只眼睛和半边嘴突然消失
  - 说到一半的话只剩气音，整个人却还站着
- 旁边的人眼睁睁看着这一幕，会比单纯死人更惊恐，因为这是“规则正在当场改写人”。
- 韩峥这一翻车必须发生在所有人面前，让所有人再也没法把陆沉的话当成疯话。
- 有人尖叫，有人崩溃后退，有人直接想跪，有人终于开始喊陆沉名字。

这一场的作用：
- 完成本章最重要的打脸
- 让主角从“被质疑”转到“被依赖”
- 把人群推向失控边缘

场景 8：所有人看向陆沉

内容要点：
- 韩峥出事后，十三层短暂陷入死一样的寂静。
- 周岚这时再也撑不住了，她可能是第一次用近乎求助的语气问陆沉怎么办。
- 林晚会本能站到陆沉身边，这个动作本身就是一种态度转变。
- 其他幸存者虽然还没完全信服，但已经默认陆沉是唯一可能带他们活下去的人。
- 陆沉自己其实也慌，但他必须在这一刻硬顶上去。
- 章尾不要让他马上展开完整指挥，只要让所有人的视线和希望同时落到他身上就够了。

章尾钩子：
建议卡在一句带压力的问话上，比如：
`所有人终于把目光都落在陆沉身上，第一次问他：现在该怎么活？`

本章节奏提醒：
- 前半章要不断封退路，不要让十三层像“可选探索”
- 中段规则必须清晰，读者要能牢牢记住三条十三层规则
- 韩峥翻车要写成公开、直观、不可辩解的打脸
- 结尾收在“主角被迫成为带路人”，为第五章接管队伍做准备

本章伏笔：
- 红鞋护士只是被规则先提到，还没真正登场
- 婴儿哭声对应的东西明显比数病房更危险
- 十三层不像普通废弃楼层，更像一个被规则侵占的夹层空间

本章读者爽点：
- 强势角色当众翻车，被规则狠狠打脸
- 主角的话第一次从“没人信”变成“所有人都不敢不听”
- 十三层正式开启，副本升级感拉满
