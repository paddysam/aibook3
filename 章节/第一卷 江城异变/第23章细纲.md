# 第 23 章细纲 错误线路

本章定位：
第二十三章要给公交副本第一阶段一个明确小回报。它不能只是“再试一次规则”，而是要让陆沉第一次真正靠 `选线` 赢一回。这个赢不是找到绝对安全车，而是提前排掉一条最错的线，把全队从一次整车级的归档里拽回来。

本章目标：
- 延续第22章的旧站牌和旧路线感
- 让陆沉第一次系统性地比较不同公交线路
- 让老李或普通幸存者承担一次更真实的现实阻力
- 让林晚继续作为现实信息辅助，补足旧线路判断
- 给出本阶段第一次真正意义上的“选路回报”
- 让 `13路` 不只是验证道具，而是整套路网的关键入口线索
- 章尾抛出 `观测线路 13` 的硬线索

本章核心冲突：
- 主冲突：到底哪条线能上，哪条线一碰就会整车出问题
- 次冲突：普通人没法理解“先排除最错的路线”这种思路，只想赶紧离开
- 恐怖来源：多条线路同时存在、旧路线表、血光强弱、线路名称和旧档案呼应

本章回报类型：
- 本章回报重点是：`主角第一次靠选线改写结果`

承接开头：
- 紧接第22章章尾
- 众人已经看见 `江城第三旧址`
- 并且知道车一旦开起来，路线接管比车厢内异常更致命

场景 1：从一辆车变成多条线

内容要点：
- 第22章结尾是旧站牌冲击，第23章开头要顺势把视角从单辆 `13路` 拉到整个公交线路网。
- 车未必会一直往前开，可以写成短暂停靠或回到某个不正常的中转站台。
- 众人这时第一次意识到：
  - 这里不是只有一辆怪车
  - 是有整套线路在同时运转
- 但要明确一点：
  - 不是抛开 `13路`
  - 而是从 `13路` 往外看，发现它只是更大线路体系里最显眼的一条
- 电子牌上会出现更多线路号，但都不完全正常：
  - 有的只有数字，没有终点
  - 有的终点是旧地名
  - 有的明明近，却血光最浓
- 陆沉开始不再盯某一条规则，而是盯整体关系。

这一场的作用：
- 把公交副本玩法从“13路验证”升级成“整套线路比较”
- 为主角第一次选线做结构铺垫

场景 2：陆沉开始拼线路

内容要点：
- 陆沉会把前面拿到的信息开始真正拼起来：
  - 旧档案里提到过的旧址
  - 地铁异常线路中的编号
  - 车窗外闪过的旧站牌
  - 当前各条公交线终点名称
- 他这时候不能像开上帝视角一样直接判断。
- 要写成他在压时间里做筛选：
  - 哪条线和旧院区关系过深
  - 哪条线像是故意把人往归档区送
  - 哪条线虽然危险，但更像是“观测线”，不是“回收线”
- 这里可以让规则视界给出辅助，但不能给完整答案。
- 比如：
  - 某两条线同样发红
  - 但一条是死红，一条是活红
- 陆沉只能先凭经验理解成：
  - 前者是吞人
  - 后者是还在运转、还留有信息
- 这里建议再补一句：
  - `13路` 很可能不是终点线
  - 而是把人送进系统后，再分配到别的线路

这一场的作用：
- 体现主角成长
- 让他的判断更像拼图，而不是作者告诉他答案

场景 3：林晚补一块现实拼图

内容要点：
- 林晚这章继续承担“现实锚点”。
- 她可以认出某一条线路的顺序完全不对。
- 比如：
  - 某条车明明应该先到老城区，再到旧院区
  - 现在顺序却倒了
- 或者她认出一块旧路线残牌，知道某条线当年就是专门跑医院周边的接驳车。
- 这块信息很关键，因为它能帮陆沉分清：
  - 哪条线只是借了旧名字
  - 哪条线真的和过去那套秩序有关
- 这段不需要长，只要让林晚显得不是跟着跑，而是能补上主角看不到的现实常识。

这一场的作用：
- 巩固林晚作为“信息副手”的位置
- 给主角的线路判断增加可信度

场景 4：现实阻力爆发

内容要点：
- 这一章必须让老李或某个普通幸存者承担一次强一点的现实阻力。
- 这个阻力不要写成反派。
- 他只是怕、急、想快点离开。
- 最合适的写法是：
  - 电子牌上有一辆车看起来更新、更亮、终点也像正常居民区
  - 老李会觉得那肯定比 `13路` 和旧站名那些车靠谱
  - 他甚至准备带两个人先上
- 这时陆沉必须拦，而且要有明确理由：
  - 那辆车亮得太干净
  - 终点写得太正常
  - 反而像最假的路
- 普通人未必能立刻听懂，所以这场冲突会比地铁时更紧。

这一场的作用：
- 把“主角成长了，但别人理解跟不上”写出来
- 让后面的回报更有打脸感

场景 5：选线成功

内容要点：
- 这一场是本章核心回报。
- 陆沉最终不是找到了完全安全车，而是成功拦下大家没去碰那条“最像正常”的线。
- 最适合的验证方式是：
  - 那辆看似最安全的车先一步发车
  - 电子牌上的终点站在发车后不断变化
  - 最后整条线路名直接从发车牌上消失
- 或者更狠一点：
  - 发车牌上那一行线路信息，连同上车人数一起被抹掉
  - 像是整辆车从系统里被直接归档了
- 众人这时才真正意识到，陆沉不是在乱赌。
- 他是第一次靠“提前判断路线”把整队从更大的坑里拉开。
- 这场回报的重点要再强调：
  - 他不是选中了最安全的
  - 而是先避开了最像“假活路”的那条

这一场的作用：
- 给公交副本第一阶段明确小胜
- 把“主角会选线”做成可见回报

场景 6：旧路线表

内容要点：
- 经历了刚才那次回避后，队伍会短暂听话。
- 陆沉趁这个空档去翻调度台、抽屉、旧记录。
- 这里不要写成平静查资料，要带一点“刚刚避过一劫”的紧迫感。
- 在最底层抽屉里，他翻出一张泛黄的旧路线表。
- 这张表的材质、印刷格式都和现实公交文件类似，但最上面多了一行不该有的红字。
- 红字内容就是：
  - `观测线路 13`
- 这一句的价值在于：
  - 它把 `13` 从数字钩子，推进成线路权限或线路类别
  - 也让公交副本从“规则副本”升级成“观测系统分流环节”
- 最好再加一层咬合感：
  - 让陆沉意识到，`13路` 和 `观测线路 13` 不是巧合重复
  - 而是同一套系统里“对象编号”和“线路编号”的第一次重叠

章尾钩子：
- 调度台最底层抽屉里，压着一张泛黄旧路线表。
- 最上方印着一行红字：
  - `观测线路 13`

本章节奏提醒：
- 第23章一定要给“选线成功”的明确回报，不能还停在试探期
- 现实阻力要真实，不要把老李写成纯拖后腿工具人
- 章尾的 `观测线路 13` 要有升级感，像比普通规则更高一层的信息

本章伏笔：
- `13` 不只是编号，也是线路分类或观测权限的一部分
- 最像正常的路线，可能反而最假
- 旧秩序和现在线路仍在按某种规则对应
- 调度台里还有更完整的系统记录可挖

本章读者爽点：
- 主角第一次不是靠临场救火，而是靠提前判断赢一把
- 普通人的误判被当场打脸
- `观测线路 13` 是很强的新线索钩子
