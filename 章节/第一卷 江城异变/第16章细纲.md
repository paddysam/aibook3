# 第 16 章细纲 封存病历

本章定位：
第十六章是第一卷里第一次真正意义上的“真相回收章”。它不是大怪谈冲刺章，而是让主线线索第一次落地的章节。重点要从旧档案室里抠出 `十年前到底出了什么事` 和 `L-13` 为什么不是随机编号。

本章目标：
- 正式进入旧档案室
- 让封存病历、旧病区名单、搬迁通知彼此印证
- 第一次回收“旧院区封闭”的事件真相
- 让 `L-13` 从可疑编号变成硬线索
- 继续推进 307、十三层、旧院区之间的连接
- 章尾给出让陆沉真正发冷的病区名单信息

本章核心冲突：
- 主冲突：档案室里哪些是真记录，哪些是被故意删改过的
- 次冲突：幸存者在档案室外是否还能稳住
- 恐怖来源：旧纸档、删改痕迹、病区名单、被人为抹掉的记录

本章回报类型：
- 本章回报重点是：`事件真相回收`

承接开头：
- 紧接第15章
- 门口封条下已经抽出一张写着 `陆沉` 的病历卡残片
- 队伍刚意识到：现实记录里真的有陆沉

场景 1：旧档案室开门

内容要点：
- 陆沉不会粗暴破门，而是先研究锁和封条。
- 这里要写出“有人很想把门封住，但封得并不彻底”的矛盾感。
- 可以安排：
  - 锁本身锈得很厉害，但链条像是后加的
  - 某一层封条底下压着更旧的封条
  - 门缝内侧有被人从里面顶过的痕迹
- 最终开门方式应尽量现实：
  - 铜锁坏了
  - 或锁芯早就不能用了
  - 真正阻止人进去的，其实不是锁，而是“心理上的忌讳”

这一场的作用：
- 让档案室从“神秘地点”变成能进入的真实场景
- 增强“这里被反复处理过”的可信度

场景 2：档案室的第一眼

内容要点：
- 旧档案室的气质要和十三层、地铁站完全不同。
- 它的恐怖来源不在即时袭击，而在被尘封的真实痕迹：
  - 一排排铁柜
  - 泛黄卷宗
  - 纸页受潮后翘起
  - 年份标签被撕掉一半
  - 某些抽屉被抽空，只剩文件夹边角
- 陆沉用规则视界看这里，不要出现太多直接死法，而是出现“记录被人为抹掉”的异常红痕。

这一场的作用：
- 强化“现实记录本身就是恐怖源”
- 把节奏从副本逃杀切到线索回收

场景 3：十年前的封楼不正常

内容要点：
- 陆沉、林晚先找到旧院区封闭通知、搬迁记录、物资转运单之类现实材料。
- 这些东西单看都像正常行政文件，但拼起来就不正常：
  - 搬迁时间太仓促
  - 某一层或某一病区在文件里被刻意跳过
  - 人员转移名单和病人名单对不上
- 这里可以让林晚和周岚一里一外配合：
  - 林晚认文件格式
  - 周岚补一嘴老护士传闻
- 最终得出一个第一轮硬判断：
  - 十年前旧院区封闭不是普通搬迁
  - 是为了压住某次无法公开说明的异常事件

这一场的作用：
- 给第一条事件真相回收
- 让读者确认医院的异常不是血月当天才有

场景 4：307和十三层的连接

内容要点：
- 接着查到更具体的病区资料。
- 重点不能全讲透，但要让读者首次清楚感到：
  - 307不是普通病房编号
  - 十三层也不是凭空多出来的鬼楼层
- 比较适合的方式：
  - 旧记录里存在某个被抹掉的病区编号
  - 这个病区和 `13`、`307` 有重叠、拆分或映射关系
- 主角得出的判断不要下死结论，而是：
  - 十三层和307像是同一场旧异常，被拆成了两种规则表现

这一场的作用：
- 把前10章的两条怪谈主线第一次串起来
- 为后面继续回收大伏笔打底

场景 5：L-13第一次落地

内容要点：
- 这一章最重要的回报，是 `L-13` 不能只当怪编号。
- 陆沉应该在病区名单、封存记录或病人索引卡里第一次看到：
  - `病人编号：L-13`
  - 或 `特殊病例：L-13`
- 但不要直接写成“这就是陆沉”。
- 最好的效果是：
  - 读者一眼就觉得这编号和他有关
  - 但角色暂时还不能完全证实

这一场的作用：
- 把档案室线索真正钉住
- 给第17章身份压迫和半揭做准备

场景 6：章尾硬钩子

内容要点：
- 最后不能收在普通总结上，要收在一个足够刺人的档案结果上。
- 最佳选择就是病区名单、封存索引卡或病人编号表。
- 陆沉看到那串编号时，不一定当场崩，但必须有“第一次被现实记录真正撞一下”的感觉。

章尾钩子：
- 一份十年前的封存病区名单上，赫然写着：
  - `病人编号：L-13`

本章节奏提醒：
- 本章重点不是死人，而是查到东西
- 但不能写成坐着翻资料，要多用“抽屉、标签、封条、被撕掉的页角、对不上号的记录”这些现场感
- 回收真相只回收一半，千万别在这里把陆沉身份一次说透

本章伏笔：
- L-13 和陆沉一定有关，但关系仍未证实
- 307与十三层本属同源异常
- 十年前的封楼事件是整部书第一层大真相

本章读者爽点：
- 主线真相第一次真正落地
- 不是怪谈吓人，而是现实记录开始证实怪谈
- L-13 的出现会明显拉高第17章追读
