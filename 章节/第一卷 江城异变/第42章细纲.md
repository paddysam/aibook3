# 第 42 章细纲 回床通知

本章定位：
第四十二章要把 `回床` 机制立住。第41章已经把“观察席余震”和 `特护3区，回床通道开启` 这条线炸出来了，这一章就不能只是坐电梯下去看新场景，而必须让读者明确明白：系统在失去对 `L-13` 的稳定锁定后，开始尝试把那些被擦过边、还没彻底洗干净的人送回对应床位。它不是要立刻收走所有人，而是在重启一套更旧、更接近医院源头的处理方案。

本章目标：
- 接住第41章章尾的 `回床通道开启`
- 正式立住 `回床` 这套新机制
- 让电梯和等待区、闸机、记录席形成明确差异
- 安排一轮和床号有关的硬验证
- 继续带老李、刘成这两条余波，但不能让两个人一起爆
- 让林晚确认 `特护3区` 不是误闯，而是医院源头回潮
- 章尾把 `特护3区` 门牌和病区走廊真正落地

本章核心冲突：
- 主冲突：众人到底该不该进入这部明显是“回床通道”的旧转运电梯，以及进去后该怎么避免被某个床号认领
- 次冲突：队伍刚刚经历“观察位断线”，现在又要面对“回床机制”，普通人会天然想逃、想拖后、想把风险往擦过边的人身上推
- 恐怖来源：推床电梯、床号播报、膝盖以下的镜面、推床声靠近、名字和床号的错位对应

本章回报类型：
- 本章回报重点是：`回床机制正式立住`

承接开头：
- 紧接第41章结尾
- 旧病床转运电梯已经亮起：
  - `特护3区，回床通道开启`
- 老李的 `09` 残印和刘成的灰白手指都被观察盲区刺激过，说明他们现在是更容易被补锁的人

场景 1：这不是人坐的电梯

内容要点：
- 开头先写这部电梯和正常医院电梯的差异。
- 最合适的外观是：
  - 轿厢又窄又长
  - 中间没有正常站位区
  - 地面有固定病床轮的卡槽痕
  - 轿厢四角还残留着旧消毒液腐蚀后的暗斑
- 更诡异的一点要落在镜面：
  - 镜子不是正常装在头顶或正前方
  - 而是只嵌在膝盖以下那一截
- 这会让读者一眼知道：
  - 这地方不是给活人照镜子用的
  - 是让“被送上床的人”确认下半身位置的
- 周岚会本能不敢进去。
- 老李则会问出一句非常普通、但很有代入感的话：
  - 这玩意儿要把谁送回去

这一场的作用：
- 把 `回床通道` 的气质立住
- 和前面所有接入 / 等待 / 记录空间形成明显差异

场景 2：回床规则

内容要点：
- 旧电梯门刚开，里面的喇叭就要给第一轮规则。
- 规则建议控制在 3 条：
  - 听见床号时，不要回答名字
  - 推床声靠近时，不要站在电梯门正中
  - 电梯内镜面只看膝盖以下
- 规则一定要写出“医院秩序感”：
  - 不是怪谈在追你
  - 而是旧护理流程在认人
- 林晚在这章可以立刻意识到一个很值钱的现实逻辑：
  - 床号是病区处理序
  - 名字只是人自己的习惯叫法
  - 这套系统要的是把你送回对应“床位”，不一定需要认你的全名
- 这和前面的等待区不同：
  - 等待区要你自己回应
  - 回床通道则更像在把你按床号往里对

这一场的作用：
- 正式立住本章规则
- 把“回床”和前面的“等待确认”做成两个不同层级的处理流程

场景 3：谁先被床号看中

内容要点：
- 这一章不能让老李和刘成都一起出问题，不然乱。
- 最适合的承担人物仍然是刘成。
- 理由很顺：
  - 他手指灰白
  - 之前已经多次被流程擦边
  - 现在最像“待接回对象”
- 最合适的表现：
  - 电梯门关到一半时，轿厢喇叭突然冷不丁报出一个床号
  - 不是名字
  - 只是：`七床，准备回床`
- 刘成先是愣。
- 然后本能去看镜子。
- 镜面里，他膝盖以下那部分站姿，和别人不一样。
- 别人的脚还在原地。
- 只有他的裤脚和鞋尖，像已经先一步站在了某张病床边。
- 这一幕非常值钱。
- 因为它不需要怪物扑脸，就足够吓人。

这一场的作用：
- 完成第一轮与床号相关的硬触边
- 让刘成继续承担“高危样本”，但不再重复前面的同一种出错方式

场景 4：老李的作用不是送死，是确认

内容要点：
- 老李这章不能也跟着被强收，不然会抢乱。
- 他更适合承担“低强度异常样本”的确认作用：
  - 当电梯报出床号
  - 或镜面开始变化
  - 他手腕上的 `09` 残印会突然像被针扎一样发热
- 这能让陆沉迅速判断：
  - 这地方确实和“被系统擦过边的人”有关
  - 但现在重点锁的是刘成，不是老李
- 老李的价值在这里是：
  - 用自身余波帮主角确认系统锁定方向
  - 同时让普通读者感到“他没被收，也不是没事”

这一场的作用：
- 不让老李的余波掉线
- 也避免他和刘成在同一章承担重复功能

场景 5：林晚确认特护3区

内容要点：
- 电梯下行过程中，林晚要给一个很值钱的现实判断。
- 最适合的方式是：
  - 她在轿厢里看到一张极旧的病区转运单角
  - 或床位顺序标注
  - 确认这套病床号和旧院区病区布局是对得上的
- 她可以直接说出：
  - `特护3区` 这个名字不是误导
  - 他们确实正在回到医院旧源头
- 这句话很重要。
- 因为它把第41章的新危机，重新扣回医院主线。

这一场的作用：
- 让这一章不只是规则验证，还承担主线衔接
- 继续保证林晚不是只负责尖叫或拉人

场景 6：病区门开

内容要点：
- 章尾必须落成具体病区，不然这章会显得像过渡。
- 最合适的章尾画面是：
  - 电梯在地下最深一层停住
  - 门缝一开，先不是人影
  - 而是一股极冷、带着旧药味和潮腥味的风压
  - 接着露出一条极窄的旧病区走廊
- 墙上的褪色牌子只剩四个还能认出的字：
  - `特护3区`
- 不要再多解释。
- 到这里收住就够。

章尾钩子：
- 电梯门在地下最深一层“叮”地一声打开。
- 门外是一条极窄的旧病区走廊。
- 墙上的褪色牌子，只剩四个还能辨认的字：
  - `特护3区`

本章节奏提醒：
- 第42章不是高潮章，重点是“新机制立住”
- 回床规则要和等待区规则明确区分，重点是床号和床位，而不是“已确认”
- 刘成继续承担高危样本，但触发方式必须和前几章不同
- 章尾必须真正落到病区，不然下一章会起不稳

本章伏笔：
- 回床通道不是普通电梯，是病区处理流程的一部分
- 特护3区现在仍在运转，不是旧档案里的死名词
- 刘成更容易被床位认领，说明他后面仍是高危擦边样本
- 老李的 `09` 仍在作为低强度确认信号生效

本章读者爽点：
- 新机制足够新鲜，和前面等待区、闸机、记录席完全不同
- “镜子只照膝盖以下”和“床号认人”都很有画面感
- 章尾落到 `特护3区`，医院源头线会有很强追读拉力
