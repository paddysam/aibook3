# 第 35 章细纲 等待归档

本章定位：
第三十五章要把 `归档` 第一次真正具象化。前面第31到34章已经完成了“入口立住、过闸验证、编号对象痕迹出现”，这一章就不能再只是更深一层地查，而要让读者真正知道：`归档` 不是一句威胁话，也不是简单死亡公告，而是一套正在运行、会把活人慢慢变成“这里一部分”的流程。最重要的是，本章必须给一次非常直观的硬验证，而且这次验证最好带上林晚的一次明显高光。

本章目标：
- 承接第34章拿到的旧物和笔迹压迫
- 把 `归档` 从概念正式变成流程场景
- 立住“等待区”的秩序恐怖，而不是怪物恐怖
- 安排一次假叫号骗人入座的硬验证
- 让读者直观看到：被确认的人会像旧照片一样开始褪色
- 承接老李被 `09` 残片短暂认领后的余波，证明“被编号碰过一次”不会立刻清零
- 让林晚先一步听出假叫号节奏不对，完成一次有效高光
- 让周岚、老李第一次真正意识到：这里最可怕的不是死，而是被安静地留下
- 章尾用 `L-13` 金属牌把身份压迫再推进一步

本章核心冲突：
- 主冲突：在一片像正常候车厅的等待区里，众人到底该怎么判断什么位置能站、什么广播能听、什么名字能认
- 次冲突：系统已经越来越明确地点陆沉，队伍里会有人开始本能觉得，只要离他远一点，也许就不会被连带写进去
- 恐怖来源：空着却显示 `已确认` 的座位、反复核对姓名的广播、看起来像在等车却不该还存在的身影、假叫号

本章回报类型：
- 本章回报重点是：`归档流程第一次实体化 + 林晚第一次明显高光`

承接开头：
- 紧接第34章章尾
- 众人已经从 `失物招领` 里确认：
  - 前编号对象不是抽象数字
  - 他们留下来的残片还会继续认活人
- 陆沉还在承受“那笔迹和自己太像”的心理压迫，但不能停下来消化
- 老李手腕上还留着一圈极淡的红色勒痕。
- 他嘴里那个不属于自己的旧站名虽然压下去了，但靠近某些区域时，喉咙里还会本能地发紧

场景 1：真正的等待区

内容要点：
- 从 `失物招领` 再往里走，空间气质要再次切一下。
- 这里不能写成阴森脏乱的鬼屋大厅。
- 最合适的是：
  - 一排排金属长椅整齐摆着
  - 屏幕反复跳号
  - 头顶广播每隔一会儿就提醒核对信息
  - 边角还立着“请按序等待”的旧牌子
- 最瘆人的地方在于：
  - 座位区明明看起来很空
  - 但有些座位前的小屏已经显示 `已确认`
  - 有些位置甚至还摆着一杯早该干掉的旧水
- 周岚会在这一场第一次明显失语。
- 因为这里太像正常等待区了。
- 正常到让人本能想坐下、想喘气、想按它的秩序来。
- 而陆沉会立刻意识到：
  - 这才是最危险的地方
  - 这里不是在等车
  - 是在等“谁先被确认”

这一场的作用：
- 把 `归档等待区` 的空间感立住
- 把本章的恐怖核心从“看见怪”切到“按秩序被收下”

场景 2：等待规则

内容要点：
- 众人刚进入，头顶广播会发出一次比前面更清晰的提示。
- 最适合的规则呈现方式是：屏幕和广播同时给，但不完全重合。
- 规则建议控制在 3 条：
  - 屏幕亮起自己的编号前，不要坐下
  - 听见“请核对姓名”时，不要回答本名
  - 等待区人数如果突然多出一人，不要第二次确认
- 这几条规则一定要让读者看出：
  - 这里不是防怪
  - 是防“被归档流程接住”
- 老李会本能提出最现实的问题：
  - 那要是叫到名字，不应也不坐，能怎么办
- 这个问题很重要，因为它正是普通人在秩序空间里最容易被逼出来的焦虑。
- 老李这时还可以承担一个很值钱的异常样本作用：
  - 他手腕上的残印在某片 `已确认` 座位区前会突然发冷或轻轻跳痛
  - 这说明旧编号残片不是撕下来就算结束
  - 被系统识别过一次的人，后面仍然可能被继续补写
- 陆沉不会立刻回答完整方案。
- 他更适合先观察：
  - 哪些屏幕刷新最快
  - 哪些广播比屏幕慢半拍
  - 哪些 `已确认` 的座位旁边，其实什么都没有

这一场的作用：
- 正式亮出本章局部规则
- 把等待区的压迫变成“每一步都像流程陷阱”

场景 3：假叫号

内容要点：
- 本章必须有一次非常直观的硬验证，而且最好来得很快。
- 最合适的方式是：
  - 一个普通幸存者听到广播里清清楚楚叫出了自己的名字
  - 广播语气标准、机械，像医院叫号一样毫无情绪
  - 同时离他最近那排空座位前的小屏也亮了一格
- 这就是最危险的地方：
  - 它看起来太像“终于轮到你”
  - 太像普通秩序
- 那个人会本能觉得：
  - 既然叫到我了
  - 那我不坐，反而更危险
- 他刚迈出去，陆沉还没完全来得及反应。
- 真正先听出不对的人，最好是林晚。
- 她要察觉到：
  - 这次广播和前面几次不一样
  - 节奏更近、尾音更短
  - 像是“临时补出来的假叫号”
- 她必须先出声喝住，再和陆沉一起冲过去。

这一场的作用：
- 给林晚一次非常明确的“不是只会跟上，而是先一步判断出来”的高光
- 为本章最硬的验证蓄力

场景 4：像旧照片一样褪色

内容要点：
- 那个幸存者即使被喝住，也最好已经慢了半步。
- 最合适的后果是：
  - 他还是坐上了半边椅子
  - 或者屁股刚沾到座位边缘
- 下一秒，最直观的归档现象就出现了：
  - 他的衣角先褪色
  - 手指边缘像被旧照片洗掉了一层
  - 半边脸的光感开始发白、失真
- 与此同时，对应小屏上的空白状态瞬间跳成：
  - `已确认`
- 这一幕一定要写得非常安静。
- 越安静，越能把“被系统收进去”这件事写得可怕。
- 陆沉负责判断怎么把人拖离座位。
- 林晚负责真正冲上去拉。
- 她必须顶着广播继续补叫名字的压迫，把那个人硬拽下来。
- 最终效果不能写成全无后遗症：
  - 人能救回来
  - 但衣袖边缘还会保留一段褪白
  - 像一小块没洗回来的旧相纸

这一场的作用：
- 完成本章最实在的硬验证
- 把 `归档` 从抽象词变成肉眼可见的人身变化
- 给林晚一次真正能让读者记住的执行高光

场景 5：最可怕的不是死，是留下

内容要点：
- 人被拖回来后，队伍情绪一定要垮一层。
- 周岚和老李在这里要承担足够真实的普通人反应：
  - 他们不再只是怕下一条规则是什么
  - 而是第一次明白，这里不是“你会不会死”
  - 是“你会不会被安静地留在这里”
- 对老李来说，这种恐惧会更具体：
  - 因为他已经真切感受过一次“别人的编号往自己身上贴”的过程
- 这时可以顺手给一点很轻但很瘆人的背景变化：
  - 远处某个已经显示 `已确认` 的空座位，扶手微微往下一沉
  - 或某个根本看不见人的位置，传来一声布料轻轻摩擦椅背的声音
- 这样不用直接让“等待的人”转头，也能让读者知道：
  - 这里的确认，不是屏幕乱跳
  - 是真的有东西被留下了
- 队伍里会开始有人更明显地和陆沉拉开距离。
- 因为普通人会本能觉得：
  - 系统现在盯得最紧的是他
  - 离他越近，越可能跟着被补进去

这一场的作用：
- 把本章恐怖从单次验证，升级成持续秩序压迫
- 把队伍的人性裂缝继续往后推一寸

场景 6：13号对象，补全确认中

内容要点：
- 众人还在消化刚才那一幕时，等待区最深处一块原本黑着的总屏突然亮了。
- 这次不能再是模糊暗示，要更直接一点。
- 屏幕一开始像正常跳号。
- 跳了两下后，所有信息一起被抹掉，只剩一行字：
  - `13号对象，补全确认中`
- 这一行字出来时，最好再配一个非常短的广播尾音：
  - `请就位。`
- 这一瞬间，所有人都会下意识看向陆沉。
- 这一下的情绪重点不是神秘，而是尴尬、压迫、被群体目光顶住。
- 因为从这时候开始，队伍里很多人会第一次真正把“系统在等他”这件事放到表面上想。
- 但本章不要在这里立刻引爆公开冲突。
- 先把这一刀压住。
- 最后让林晚或陆沉在深处那张空椅背后，发现一块旧金属牌：
  - `L-13`
- 它的存在说明：
  - 这个位置，曾经真的长期属于某个 `L-13`

章尾钩子：
- 等待区最深处那张空椅背后，嵌着一块旧金属牌。
- 上面刻着：
  - `L-13`

本章节奏提醒：
- 第35章必须把 `归档` 变成能看见的流程，不能再只停在概念层
- 假叫号那一段一定要写得“太像正常秩序”，这样才更吓人
- 林晚这章一定要有存在感，她的高光不是打怪，而是先听出不对、硬把人拉回来
- 章尾的 `L-13` 既是身份压迫，也是下一章记录席的自然接力点

本章伏笔：
- 等待区的 `已确认` 说明归档不是瞬间完成，而是有确认步骤
- 假叫号证明系统会主动制造“你以为轮到你了”的秩序陷阱
- 队伍里有人已经开始本能和陆沉拉开距离
- `L-13` 这个位置很可能曾经真实存在过长期占位对象

本章读者爽点：
- 归档终于不是抽象概念，而是出现了非常直观的人体褪色画面
- 林晚第一次拿到一段很能立角色的硬执行高光
- 章尾 `L-13` 金属牌会把身份线和场景线狠狠干到一起
