# 第 29 章细纲 选错的人

本章定位：
第二十九章要给这一阶段一次现实层面的重击。它的重点不是再死人，而是让读者感到：主动选战场不代表万能，选错的人不会立刻死，但会被系统悄无声息地从现实里“转走”。

本章目标：
- 承接第28章疯涨的新亮点
- 让第25章就被预定的老张彻底扛不住，私自偏离路线
- 把“错误线路”从交通工具扩展到城市节点层面
- 让陆沉第一次尝试根据观测残留反推错误线路的去向
- 章尾让那个人从亮红变成灰黑，形成更深层的不安

本章核心冲突：
- 主冲突：当节点在变，队伍还能不能继续相信陆沉的改线判断
- 次冲突：普通人最终还是会在极限压力下做出错误选择
- 恐怖来源：错路、错层、错位、被系统安静收走

本章回报类型：
- 本章回报重点是：`现实打击 + 主角能力再升一格`

承接开头：
- 紧接第28章章尾
- 一个原本极弱的点，忽然疯涨成血红

场景 1：疯涨的点像答案

内容要点：
- 疯涨的血点会天然给队伍一种“系统在提示正确答案”的错觉。
- 这正好和第27章“最亮的未必最真”继续呼应。
- 陆沉会更谨慎，不会立刻过去。
- 但别人不一定受得了继续等。

这一场的作用：
- 给后面“选错的人”制造诱因

场景 2：有人扛不住了

内容要点：
- 这个人不能写成坏人。
- 最适合的就是第25章已经被“错误线路预定”的老张。
- 这样这一轮打击就不是新起一条线，而是前面压了几章的后果终于兑现。
- 他会因为：
  - 看到亮点疯涨
  - 听到某句错位指引
  - 或觉得“再不走就全完了”
  而做出一个看似非常合理、但实际上错误的选择。
- 他不是突然狂奔，而是先悄悄偏了一步，再越走越远。

这一场的作用：
- 让“选错的人”更真实，而不是强行送死

场景 3：错层开始

内容要点：
- 最可怕的不是他走远，而是他还在大家视线里。
- 只是：
  - 站位越来越怪
  - 身后的景变得和这边不一样
  - 路灯颜色不对
  - 声音像隔着一层玻璃
- 众人会第一次明确感觉：
  - 他不是离开了
  - 是走进了另一条错误线路

这一场的作用：
- 把“错误线路”从抽象概念变成直观看见的错层

场景 4：还能回应，却回不来了

内容要点：
- 这章最阴的地方就在这里。
- 那个人还能回头、还能说话、还能看见他们。
- 甚至会说：
  - `这边是对的`
  - `你们快过来`
  - 或 `这里什么都没有`
- 但他的声调、口型、动作，已经开始和现实错位。
- 林晚和周岚这时会非常害怕，因为这是比立刻死人更难受的一种状态。

这一场的作用：
- 把“人还在，但已经不在同一层”写实
- 给本章最大的情绪压迫

场景 5：陆沉第一次反推错误线路

内容要点：
- 陆沉不会冲过去救。
- 这点很关键：不是因为冷血，而是他已经明白，错层一旦形成，直追过去只会再多一个。
- 但他也不是眼睁睁看着。
- 这章他的成长点是：
  - 第一次根据观测残留、亮点变化、空间错位去反推“这条错线通向哪”
- 这里最好再吃到前两章埋下的前置证据：
  - 他会发现错层形成后，观测残留和现实错位之间也有极短滞后
  - 系统像是在“更新这个人已经被转走”的状态，但还没完全写完
- 这就是第30章能误导系统的核心基础之一。
- 他不需要完全算出来，只要得出一个有价值的判断：
  - 错误线路不是随机吃人
  - 它会把人送往某种“归档层”
  - 而那个人现在还没彻底消失

这一场的作用：
- 让主角从“会避死”升级到“会反推系统走向”

场景 6：名字褪色

内容要点：
- 规则视界里，那个人原本还发亮的名字开始变色。
- 先是红。
- 再慢慢发灰。
- 最后接近发黑。
- 这比当场死更吓人，因为这像是在看一个人被“注销”。
- 这也是最适合收章尾的地方。

章尾钩子：
- 在规则视界里，那个人的名字，正一点点从亮红褪成灰黑。

本章节奏提醒：
- 第29章不能写成又一个普通送死章
- 错路最可怕的地方是“人还在，但已经不在同一层”
- 最好明确就是老张出事，这样第25章到这里会形成完整回收
- 主角的回报点是 `第一次反推错误线路`，不是救下人

本章伏笔：
- 灰黑化不是死亡本身，而可能是被归档前的过程
- 错误线路通向的层，和后面更深的主网节点有关
- 系统不是只会记录，还会做“注销/归档/转移”
- 系统写入“人已被转走”的状态时，可能存在很短的判定空档

本章读者爽点：
- 现实层打击足够重
- 错层感非常新鲜
- 主角能力在“看系统走向”这件事上又升了一格
