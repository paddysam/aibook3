# 第 14 章细纲 终点站不存在

本章定位：
第十四章要给地铁副本一个阶段性突破。重点不是彻底通关，而是让陆沉从“在副本里活下来”升级到“看懂副本正在把人送去哪里”，并借此拿到一条能把主线拉回医院旧档案室的硬线索。

本章目标：
- 把“终点站了”这句私人呼唤接实
- 让陆沉意识到地铁副本的真正危险是 `线路目的地`
- 第一次发现站名、编号、旧区/医院之间有对应关系
- 完成一次地铁副本内的小突破
- 给队伍一个阶段性脱离地铁站的机会
- 章尾把“旧档案室必须查”钉死

本章核心冲突：
- 主冲突：列车和站台，到底哪边更危险
- 次冲突：终点站规则和现实线路图严重冲突
- 恐怖来源：站名重复播报、线路图异常、终点站不存在

本章回报类型：
- 本章回报重点是：`副本阶段突破 + 主线线索到手`

承接开头：
- 紧接第十三章章尾
- 车厢深处传来：
  - `陆沉，到终点站了。`
- 队伍对地铁副本的恐惧已经被彻底拉满

场景 1：那句话是谁说的

内容要点：
- 那句“到终点站了”响起之后，队伍会本能地再次乱一瞬。
- 但这次不适合再死人，而是让读者和主角一起去追这个声音：
  - 是广播？
  - 是车厢乘客？
  - 还是副本直接在点陆沉？
- 陆沉要先确认一件事：
  - 这句话不是站内广播
  - 因为声音太近，也太像故意说给他一个人听
- 林晚会立刻意识到：
  - 这和医院里的点名一样
  - 地铁副本也已经把陆沉列为重点目标

这一场的作用：
- 把章尾一句呼唤变成持续压力
- 强化“陆沉被不同副本共同点名”的主线

场景 2：站名重复播报

内容要点：
- 就在众人试图重新判断下一步时，站台广播开始播报站名。
- 第一次播报正常。
- 第二次播报和第一次完全相同。
- 第三次播报前，规则视界会突然把那条规则亮得极红：
  - `如果同一站名被连续播报三次，立刻离开站台边缘。`
- 这里重点不是死人，而是“时间被卡住了”的恐惧。
- 陆沉要立刻带大家离开站台边缘，避开这一轮整站级危险。

这一场的作用：
- 验证另一条地铁规则
- 让“站名”这件事成为后面破解线路的钥匙

场景 3：线路图不对

内容要点：
- 站名播报过后，陆沉会把注意力转到站台线路图、列车显示屏、站名牌上。
- 他会发现几个极不正常的地方：
  - 当前站点不在正常地铁图上
  - 终点站名字和显示屏滚动内容对不上
  - 某个站点编号反复出现，但地图上本不存在
- 这里可以给一个非常关键的视觉线索：
  - 某个编号和医院旧区告示、旧病历编号、十三层标记产生对应
- 这一发现不用完全解释，但要足够硬。

这一场的作用：
- 把地铁副本从“站台求生”推进到“线路异常”
- 让读者知道，这个副本不是随机拼出来的

场景 4：终点站根本不存在

内容要点：
- 陆沉交叉比对广播、线路图和规则视界后，做出一个阶段判断：
  - 这趟列车真正的终点站，根本不在正常线路里
  - 如果上了那趟“对的车”，也未必会到现实世界
- 他要进一步意识到：
  - 地铁副本不是在送人回家
  - 是在把人运往某个“被规则吞掉的地方”
- 这时可以让某个显示屏短暂闪出旧站号、废弃站名或编号：
  - 比如 `13`、`L-13`、旧院区病区编号之类
- 不要一下说透，只让陆沉意识到：医院和地铁通过编号相连。

这一场的作用：
- 给本章主线认知升级
- 为回档案室建立必要性

场景 5：副本内小突破

内容要点：
- 陆沉不会完整打穿地铁副本，但会完成一次阶段性突破。
- 比较适合的处理：
  - 他不再试图“上车”，而是带大家利用广播和站台规则，避开错误列车
  - 或反过来利用列车停稳/不开门的时间差，撤出站台核心区域
- 这一轮回报的关键不是杀怪，而是：
  - 他们终于没被列车带走
  - 地铁副本第一阶段没有把他们吞进去

这一场的作用：
- 给读者一个“撑过来了”的阶段感
- 不至于让副本一直只压不回报

场景 6：必须回医院

内容要点：
- 地铁副本小突破后，陆沉拿到关键线索：
  - 某个站号或终点站编号和医院旧区/档案中的东西一致
- 林晚立刻反应过来：
  - 这不是巧合
  - 医院旧档案室必须回去查
- 这时候“回医院”不再像绕回去，而像顺着线索去找源头。
- 队伍内部可以有人崩溃：
  - 好不容易出来了，为什么还要回去
- 但没人能给出更好的方向。

章尾钩子：
- 最后收在一条够硬的线索上最稳：
  - 陆沉盯着站台旧线路图上的一个编号，低声说：
  - `这个编号，我在医院旧区见过。`

本章节奏提醒：
- 本章不能又变成纯站台逃命
- 要把“地铁线路本身有问题”这层推出来
- 不要完整通关地铁副本，留一点余味和未解压力
- 章尾必须把主线重新拉回旧档案室

本章伏笔：
- 终点站与旧院区、十三层或病历编号有关
- 地铁副本只是城市节点的一部分，不是全部
- 陆沉被点名这件事已经跨副本成立

本章读者爽点：
- 地铁副本第一次阶段突破
- 主角不只是活着，还看懂了规则“要把人送去哪”
- 医院主线被更硬地拉回来
