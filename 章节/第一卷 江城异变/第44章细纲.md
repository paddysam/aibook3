# 第 44 章细纲 空床位

本章定位：
第四十四章要把“回床”从规则层压到身体层。前面第42到43章已经立住了回床通道和特护3区的病区秩序，这一章就不能再只是继续认识病区，而必须让读者切身感受到：等待区的椅子会确认人，病区的床位也会确认人，只是它确认得更像一套“合理治疗流程”。这章的回报不是大反制，而是让陆沉第一次真正把“等待槽位”和“床位落点”连起来。

本章目标：
- 承接第43章床位屏上的 `L-13：接回失败，继续观察`
- 把空床位的危险从概念写成身体反应
- 让刘成继续承担高危样本，但触发方式不能重复前几章
- 让陆沉第一次明确 `等待区 = 流程前厅 / 病区床位 = 实际落位区`
- 章尾给出 `接回名单`

本章核心冲突：
- 主冲突：众人怎么在一排空床里判断哪一张只是空着，哪一张已经在“等人”
- 次冲突：看见床位会主动认人之后，队伍会进一步崩溃，普通人会天然觉得这里根本没有可落脚的地方
- 恐怖来源：床头小屏、床垫凹痕、输液架、冷区、床号和人体状态的对位变化

本章回报类型：
- 本章回报重点是：`床位认领机制坐实`

承接开头：
- 紧接第43章章尾
- 众人刚看见：
  - `L-13：接回失败，继续观察`
- 这说明特护3区里真的存在和 `L-13` 对应的长期床位

场景 1：空床不是空的

内容要点：
- 开头不要急着让人出事，先让床位“活起来”。
- 最适合的方式是：
  - 众人沿着病区继续往里走
  - 左右两侧是一张张整齐的空床
  - 有的床头挂着记录夹
  - 有的床边立着输液架
  - 有的床单甚至压得一丝不苟
- 最瘆人的地方在于：
  - 有些床头灯明明灭着
  - 床垫中央却留着长年压出来的凹痕
  - 像刚有人从床上起身，却没走远
- 老李最贴普通人的反应是：
  - 他会刻意不去看床
  - 因为越像正常病房，他越觉得只要看久一点，就会有哪张床把自己当病人

这一场的作用：
- 把“空床位”做成一种新的主动危险
- 继续累积病区秩序咬人的压迫感

场景 2：谁被床位盯上

内容要点：
- 这一章的承担人物仍然最适合是刘成。
- 理由非常顺：
  - 他在回床通道里已经被某个床号碰过
  - 手指也还留着灰白残相
  - 他最像“正在被接回一半的人”
- 最合适的第一层触发是：
  - 他经过一张床时
  - 床头小屏自己亮起
  - 先是空白
  - 然后慢慢跳出：`待接回`
- 这时他还没碰床。
- 但身体先不对：
  - 呼吸节拍乱掉
  - 肤色开始往床位那块更冷的区域靠
  - 说话时嘴唇发白，像体温被一点点往床里抽

这一场的作用：
- 让床位认人的危险在身体层面第一次坐实
- 也让刘成的“高危样本”线继续往前推

场景 3：不是吸过去，是落下去

内容要点：
- 这一章和前面的等待区一定要区分开。
- 最关键的不同点是：
  - 椅子更像“确认”
  - 床更像“落位”
- 所以这次不要写成刘成被猛吸到床上。
- 更合适的是：
  - 他站在床边时，重心一点点歪
  - 不是向前扑
  - 而像身体的某一部分已经先“落”进了床垫凹痕里
- 可以加一个很有画面的细节：
  - 床单中央没有人压上去
  - 却自己缓缓往下陷了半寸
  - 正好对应刘成此刻失温、发白的那一侧身体
- 这会让读者非常直观地感觉到：
  - 这张床不是在等一个人躺上来
  - 它是在提前把“应该落在这的人”认回去

这一场的作用：
- 把这章恐怖来源和第35章彻底拉开
- 让“落位区”这个概念成立

场景 4：陆沉的新判断

内容要点：
- 陆沉不能只救人，得在这章拿到一条主线判断。
- 最合适的是：
  - 他一边把刘成从床边拖开
  - 一边看床头屏、床垫凹痕和病区状态变化
  - 最后明确说出：
    - 等待区是流程前厅
    - 病区床位是实际落位区
- 这句话很值钱。
- 因为它会把前面的几章真正串起来。
- 同时也说明：
  - 医院线不是脱节旧线
  - 而是这一整条归档处理链最深、最实的一段
- 林晚和周岚都可以在这时接住这条判断：
  - 林晚理解逻辑
  - 周岚则理解病区秩序本身为什么这么怪

这一场的作用：
- 给本章一个真正推进主线的小回报
- 把病区功能和前面的等待区功能钉死

场景 5：接回名单

内容要点：
- 章尾必须给一个能直接推进下一章的硬物件。
- 最适合的是护士站抽屉。
- 众人刚把刘成拖离那张床，护士站抽屉里会被某次碰撞震开一条缝。
- 陆沉过去拉开后，里面压着一叠发脆的病区转运单。
- 最上面那张用红字写着：
  - `接回名单`
- 这个词的杀伤力非常大。
- 因为它说明：
  - 回床不是偶发
  - 这地方一直在有名单地接人
- 本章不要继续往下查。
- 到这里收住最值钱。

章尾钩子：
- 护士站最底层抽屉里，压着一叠发脆的病区转运单。
- 最上面那张红字写着：
  - `接回名单`

本章节奏提醒：
- 第44章重点是“床位会认人”，不是再发一轮病区规则
- 刘成继续承担高危样本，但不能写成“又是被叫名字就翻车”
- 陆沉必须拿到一条明确判断，不然本章会显得只是受压
- `接回名单` 是这章最值钱的收口，不要提前透太多

本章伏笔：
- 特护3区确实在执行一份有秩序的接回流程
- 床位落点和等待区确认之间存在完整前后链
- 刘成后面仍有继续被接回的风险
- 护士站是下一章继续查接回历史的天然入口

本章读者爽点：
- “床位主动认人”这种恐怖源很新鲜
- 主角第一次把前面几章的流程逻辑完整咬起来
- `接回名单` 这个章尾会让人很想立刻看下一章
