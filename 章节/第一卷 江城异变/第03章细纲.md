# 第 03 章细纲 第一次救人

本章定位：
第三章是前三章里最关键的“回报章”。前两章已经完成 `死人立钩子` 和 `规则展示 + 第一次避死`，这一章必须立刻给出 `第一次救人 + 第一次被正眼看待 + 十三层登场`。读者要在这一章明确感受到：陆沉不只是能自保，他已经能开始带别人活下去。

本章目标：
- 让 307 病房的规则第一次爆发
- 让陆沉完成第一次精准救人
- 让林晚成为第一个真正开始相信陆沉的人
- 让周岚和其他幸存者第一次意识到陆沉真的能看见什么
- 让 `十三层` 作为更大副本入口正式登场

本章核心情绪：
从广播后的死寂和余惊，迅速推进到新的近距离生死局，再在“救人成功”的短暂回报后，抛出更大的未知恐惧。

承接开头：
- 紧接第二章结尾
- 广播点名刚结束，陆沉刚完成第一次精准避死
- 307 病房门上的观察窗后，已经出现了那张不该存在的人脸

场景 1：广播后的死寂

内容要点：
- 广播消失后，整条走廊陷入一种比尖叫更可怕的安静。
- 所有人都亲眼看见陆沉像提前知道什么一样活了下来，但一时没人敢先开口。
- 周岚嘴上还想强撑，说也许只是巧合，可语气已经明显不稳。
- 林晚第一次认真看陆沉，她开始意识到陆沉和其他人看到的东西不一样。
- 就在这种死寂里，307 病房里再次传来敲门声，把刚松下来的一口气又拽紧。

这一场的作用：
- 承接第二章的小高潮
- 让“陆沉的话开始有人动摇”自然成立
- 把危险顺滑推进到 307 病房

场景 2：307 病房规则触发

内容要点：
- 病房门上的观察窗后，那张脸缓缓贴近，但病房登记信息和监控记录里根本没有这个人。
- 陆沉用规则视界去看，终于看到和 307 相关的关键规则再次浮现：
  - `不要和 307 病房里多出来的病人对视超过三秒`
- 这一次规则旁边最好还能浮现更短促的危险提示，比如：
  - `死亡触发：对视`
  - `结果：眼球破裂`
- 规则比前一章更完整，说明陆沉的能力已经因为避死成功而变清晰了一点。
- 众人还没意识到危险具体是什么，只是本能地盯着 307 看。

这一场的作用：
- 让 307 病房从伏笔变成眼前危机
- 让规则视界第一次出现更完整的“死法片段”

场景 3：林晚差点触发死局

内容要点：
- 林晚因为离 307 最近，出于职业本能想上前确认病房里到底有没有人。
- 她可能先是想看清观察窗，或者本能地伸手去推门。
- 陆沉一抬头，就看见林晚头顶浮现出极短的死亡倒计时和死法提示。
- 最好把这一刻写得很短很急，像一把刀突然架在喉咙上：
  - `剩余存活时间：00:00:05`
  - `死亡触发：对视`
- 陆沉根本来不及解释，也没时间让她自己反应。

这一场的作用：
- 把危险精准锁到一个具体人物身上
- 为第一次救人做爆发准备

场景 4：第一次救人

内容要点：
- 在林晚就要抬头看清观察窗的一瞬间，陆沉直接冲过去，把她硬生生拽开。
- 动作一定要狠，最好不是温和提醒，而是近乎粗暴地把她按低头、拖离门口。
- 林晚会本能地挣一下，甚至觉得陆沉疯了。
- 下一秒，她刚才站的位置就爆出异常，比如：
  - 观察窗内突然睁开一双充血的眼睛
  - 门缝里伸出一只青白的手
  - 玻璃上猛地炸开蛛网状血纹
- 林晚亲眼看见这一幕，才意识到陆沉刚才是在救她。

这一场的作用：
- 完成第三章最核心的回报点
- 给主角第一次明确“救人爽点”
- 让林晚和读者一起确认：陆沉不是运气好，他是真的能提前看到死法

场景 5：第一次被正眼看待

内容要点：
- 林晚惊魂未定，第一次不再把陆沉当普通护工看，甚至会主动追问：你到底看到了什么？
- 周岚和其他幸存者也明显不敢再像之前那样随口否定他，但仍然没有完全服气。
- 这里不要让所有人立刻倒向陆沉，只要做到“他们开始怕他说的话是真的”就够了。
- 陆沉趁这个时机第一次明确说出一句结论：
  - 这里有规则
  - 乱看、乱回、乱碰，都会死人
- 这是陆沉第一次真正开始掌握一点点话语权。

这一场的作用：
- 让主角从“被当成晦气源头”转到“开始有人听他说话”
- 给第四章、第五章进一步建立带队资格打基础

场景 6：能力第一次升级

内容要点：
- 救下林晚之后，陆沉眼前的规则文字忽然稳定了一瞬。
- 他明显感觉到：
  - 字迹没那么模糊了
  - 某些地方开始能看到更完整的规则残句
  - 别人头顶的死亡提示也比刚才更清晰
- 这种升级不要写成太强的游戏面板，更像是感知突然“开了一档”。
- 陆沉第一次明确意识到，成功避死和救下本该死的人，都会让 `规则视界` 变强。
- 这会给他一个很重要的心理变化：他不能只顾自己躲，他得主动找生路。

这一场的作用：
- 确立中短期成长逻辑
- 让主角未来“越救人越能看清规则”变得合理

场景 7：十三层亮起

内容要点：
- 危机短暂压下去后，医院的电梯忽然自己启动。
- 电梯数字在 7、8、10 之间乱跳，最后停在一个从未亮过的数字上：`13`
- 林晚声音发抖，告诉陆沉住院楼的十三层早在很多年前就封死了，按理说根本不该存在。
- 这时陆沉看向电梯方向，规则视界再次出现新的模糊提示，但比前面所有规则都更危险、更难看清。
- 比如只给出残缺感很强的一句：
  - `进入十三层后，不要……`
- 电梯门缓缓打开一条缝，里面没有人，只有冷气、血腥味，和一盏忽明忽暗的灯。

章尾钩子：
建议收在一句既具体又能直接勾出下一章的画面上，比如：
`林晚脸色惨白地说，十三层早就封死了，可电梯刚刚停在了十三层。`

本章节奏提醒：
- 开头不要慢，广播后的安静只能当过渡
- 中段必须把“救人”写成一个非常清晰的爆点
- 后段不要解释太多十三层，只负责把它立成更大的副本入口

本章伏笔：
- 307 病房里“不存在的病人”只是前菜
- 陆沉的能力会随着避死和救人而提升
- 十三层对应的规则级别明显高于前面的广播和 307 病房

本章读者爽点：
- 主角第一次精准救人
- 主角第一次被别人正眼看待
- 主角第一次明确感受到自己的能力在成长
