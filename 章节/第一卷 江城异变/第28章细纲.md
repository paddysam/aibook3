# 第 28 章细纲 熄灭的点

本章定位：
第二十八章要验证一件很重要的事：节点不是静态的地图提示，而是会变化、会吞没、会自行关闭的“活连接”。这一章的重点不是怪谈本身跳出来，而是 `地图规则开始动起来`。

本章目标：
- 解释“亮点熄灭”不是消失，而是某种状态变化
- 让陆沉第一次意识到节点会动态变化
- 给队伍一个非常直观的“退路没了”的后果
- 让主角第一次在城市层面做出动态改线
- 章尾抛出一个原本极弱、却突然疯涨的血点

本章核心冲突：
- 主冲突：刚选中的点突然没了，现在该怎么判断
- 次冲突：队伍会因为退路消失而更慌，甚至开始怀疑陆沉的判断
- 恐怖来源：节点熄灭、区域失效、地图变化、系统重新分配路径

本章回报类型：
- 本章回报重点是：`动态规则升级`

承接开头：
- 紧接第27章章尾
- 众人刚争完路线，其中一个点突然熄灭

场景 1：熄灭不是消失

内容要点：
- 众人会先本能以为：
  - 熄灭的点没了
  - 或危险降低了
- 陆沉不能马上下结论，要先看那一块区域在规则视界里的变化。
- 最终他会发现：
  - 不是没了
  - 而是从“可观测亮点”变成了“整片发黑的死区”
- 这说明熄灭的不是信号，而是“入口资格”。

这一场的作用：
- 把“亮点熄灭”定义成更危险的变化
- 让城市节点第一次有活物感

场景 2：退路消失的现实后果

内容要点：
- 这章一定要有一个现场后果，不然会只剩概念。
- 最适合的是：
  - 原本坚持想去那个点的人往前看时，发现那一整条街的轮廓都像被抹薄了一层
  - 或原本还能看到的便利店、楼道、候车亭，全都像隔上了不属于现实的雾
- 也可以更具体：
  - 某个平时能走的路口，在规则视界里直接变黑
- 重点不是炸，而是让人知道：你晚一步，这条路就不再是路了。

这一场的作用：
- 让“节点熄灭”从图上变化变成现实威胁

场景 3：队伍二次动摇

内容要点：
- 这个时候队伍会再次摇晃。
- 周岚会更怕，因为她本来就倾向保守路线。
- 老李会出现一轮短暂崩溃：
  - 觉得走哪都不对
  - 认为不如随便选一条赌
- 林晚则会本能看向陆沉，等他的判断。
- 老张这时最好也有一点更具体的不对劲：
  - 对某个方向反应异常
  - 或像是被某个点“牵”了一下
- 这不是立刻爆发，而是为第29章提前压住那根线。
- 这一章要写清楚：节点一变化，普通人对主角判断的信心也会立刻松动。

这一场的作用：
- 保持现实阻力
- 让主角后面的改线决定有重量

场景 4：陆沉第一次动态改线

内容要点：
- 这是本章主角的成长回报。
- 以前他更擅长：
  - 识别
  - 警告
  - 排除
- 这一章开始，他要学会：
  - 地图会动
  - 规则会变
  - 所以选择也必须跟着变
- 他会快速重排三个点的优先级：
  - 熄灭的点不再考虑
  - 最亮的点也许太像诱饵
  - 原本不起眼的弱点反而可能是下一个真入口
- 但这里一定要补一个关键判断，不然第29章别人会显得比主角更有道理：
  - 这个弱点虽然在变强，却不是“稳定发亮”
  - 更像是被某种外部规则突然接通、强行顶亮
  - 它亮得太快，反而像一个刚被摆出来的喂口
- 所以陆沉不会直接带队扑过去，而是会先观察它的连接方式和稳定度。
- 这里一定要写得像临场判断，不要写成事后总结。

这一场的作用：
- 把主角推进到“城市层动态判断”的新层级

场景 5：弱点疯涨

内容要点：
- 就在陆沉重新排序后，那个原本最弱、最不起眼的点突然开始迅速变亮。
- 不是慢慢亮，而是像被什么东西瞬间接通。
- 这一幕必须有强视觉冲击：
  - 先一点暗红
  - 后一秒整块疯涨成血色
  - 周边建筑的轮廓都被染亮
- 这会让主角意识到：
  - 系统不仅会关闭点
  - 也会临时接通点
- 更重要的是：
  - 它的亮法不对
  - 不是“被发现”，而像“被投喂给他们看”
- 这个变化说明，他们还在系统的筛选流程里。

章尾钩子：
- 一个原本极弱的点，忽然从地图边角疯涨成血红。

本章节奏提醒：
- 第28章一定要让“节点会动”这件事被实锤
- 重点不是开新怪，而是让地图本身活过来
- 主角的成长要落在“动态改线”上

本章伏笔：
- 熄灭的点不是废了，而可能是被系统关闭或吞没
- 突然疯涨的点会把他们引向下一轮真正的错误线路风险
- 节点变化很可能和观测系统的选择有关
- 老张身上“被错误线路预定”的后劲，还没有结束

本章读者爽点：
- 节点不是死的，这一下会非常新鲜
- 主角第一次在城市层面临场改线
- 章尾弱点疯涨，适合拉第29章
