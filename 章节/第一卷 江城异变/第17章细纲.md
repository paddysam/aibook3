# 第 17 章细纲 十年前的名字

本章定位：
第十七章承接第十六章的 `L-13`，重点不是再补事件材料，而是把真相往主角本人身上压。也就是说：第16章回答“医院出了什么事”，第17章回答“陆沉为什么会被这件事反复点名”，但只揭半层，不讲透。

本章目标：
- 让陆沉第一次在旧记录里看到和自己高度相关的硬证据
- 让“L-13”从编号变成和他本人贴脸的压迫
- 让陆沉出现一轮更深的身份震荡
- 让林晚继续承担“把主角从自我怀疑里拉回来”的作用
- 让主角明确：不能再逃，必须主动查
- 章尾用旧照片或旧签名给下一阶段再抬一层

本章核心冲突：
- 主冲突：陆沉到底是“被规则盯上”，还是本来就与旧异常有关
- 次冲突：旧记录是真是假，还是怪谈在借真档案做误导
- 恐怖来源：旧病历里的名字、对不上的时间、照片里相似的人

本章回报类型：
- 本章回报重点是：`身份线半揭`

承接开头：
- 紧接第16章
- 陆沉已看到：
  - `病人编号：L-13`

场景 1：编号后的名字

内容要点：
- 陆沉不会满足于只看到编号，接着去翻编号对应的病历卡、索引页或登记簿。
- 找到的不是完整病历，而是被抽残、被删页、被重新归档过的材料。
- 其中最关键的一条，应当直接给到：
  - `姓名：陆沉`
  - 或一个与陆沉完全一致、但时间对不上的身份信息
- 一定要制造“现实证据比怪谈写名字更吓人”的效果。

这一场的作用：
- 把第16章的硬线索往主角身上贴
- 让“门牌背后写陆沉”得到现实回音

场景 2：时间和年龄全部对不上

内容要点：
- 这一章最值钱的不是“名字一样”，而是“时间完全不对”。
- 可以设置成：
  - 病历日期早于陆沉出生
  - 年龄和现在完全对不上
  - 监护人、住址、身份栏出现异常空缺
- 这样读者会明确知道：
  - 这不是普通重名
  - 但也还不能直接认定“那就是陆沉”
- 林晚在这里的作用不是安慰，而是帮陆沉确认：
  - 这份档案不是刚伪造的
  - 纸张、格式、印章都是真的旧档

这一场的作用：
- 把“身份压迫”拉到最高
- 保持悬念，不把答案说死

场景 3：主角更深一轮动摇

内容要点：
- 这一章必须给陆沉一次更深的心理冲击，但不能写崩。
- 这轮动摇最好比第7章更私人、更沉：
  - 他开始怀疑自己缺失的不是近期记忆，而是更早的一段人生
  - 怀疑自己进这家医院做护工，是不是从一开始就不是偶然
- 但不能让他陷进去太久。
- 重点是：他第一次真的觉得“自己可能就是这场规则的一个缺口”。

这一场的作用：
- 让主角从“被怪谈针对”升级到“我可能本来就在局里”

场景 4：林晚把他拉回来

内容要点：
- 第7章林晚是用一句“至少一直在救人”把他拉回来。
- 这一章不能重复同样方式。
- 更适合的写法是：
  - 她提醒陆沉，怪谈最会用真东西骗人
  - 证据是真的，不代表解释就是真的
  - 现在最重要的不是先定义自己是谁，而是弄清楚这些记录为什么会被封
- 这样林晚就不是情绪支撑，而是思路支撑。

这一场的作用：
- 强化林晚和主角的搭档关系
- 避免主角陷入长篇自我怀疑

场景 5：旧照片

内容要点：
- 病历和名单之后，最适合再给一个视觉冲击更强的东西：旧照片。
- 旧照片上最好出现：
  - 被封病区门口
  - 模糊的人群
  - 一个和陆沉极像的人
- 不要写成一眼百分百确认，而是“极像到让人发冷”。
- 照片细节里还可以埋一个下一阶段伏笔：
  - 照片角落出现某个站号
  - 某块牌子写着 `13`
  - 某张床位卡和地铁编号呼应

这一场的作用：
- 给第17章章尾一个比编号更直观的炸点
- 为第18章走向城市规则节点继续搭桥

章尾钩子：
- 档案夹最深处掉出一张旧照片。
- 照片里的病区门口，站着一个和陆沉长得极像的人。

本章节奏提醒：
- 本章重点是“身份压迫”，不是继续讲十年前事件流程
- 时间线和年龄对不上，是本章最重要的悬念手段
- 林晚要帮主角稳住，而不是一起崩

本章伏笔：
- 陆沉与L-13之间的关系仍未定性
- 旧照片可能藏着下个节点线索
- 陆沉来医院做护工这件事，可能本来就不是偶然

本章读者爽点：
- 主角身份线第一次真正被证据压实
- 旧记录不是名字巧合，而是时间线级别的不合理
- 章尾旧照片非常适合吊下一章
