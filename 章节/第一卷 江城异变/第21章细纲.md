# 第 21 章细纲 欢迎上车

本章定位：
第二十一章的任务是把第20章章尾的 `江城规则观测点：第13号` 真正接成新阶段开局。它不能写成“刚进公交站就立刻死人”的重复套路，而要先把公交调度中心的气质立住：这里不像医院那样压迫，也不像地铁那样冰冷，它更像一个仍在正常运转的系统前厅，越正常，越让人不敢乱动。

本章目标：
- 接住第20章章尾的观测点钩子
- 让公交调度中心和医院、地铁形成明显差异
- 把 `欢迎回家` 变成新的规则入口
- 建立公交副本的核心恐怖逻辑：`不是车里有什么，而是你会被哪条线接走`
- 提前埋下 `13路` 不只是普通公交，而是和陆沉存在私人关联
- 让队伍内部出现第一轮“这里看起来反而没那么危险”的误判
- 在章尾抛出 `13路` 车辆主动亮灯的硬钩子

本章核心冲突：
- 主冲突：刚进公交调度中心，到底该不该继续往里走、该不该碰任何一辆车
- 次冲突：幸存者在经历医院和地铁后，会本能地把“看起来正常”误判成“暂时安全”
- 恐怖来源：过于正常的调度秩序、还在运行的电子牌、无人却亮灯的公交、欢迎语

本章回报类型：
- 本章不是硬爽点章
- 回报重点是：`公交副本机制正式立住`

承接开头：
- 紧接第20章章尾
- 陆沉已经看见：
  - `江城规则观测点：第13号`
- 众人刚刚拿到“主角和编号系统硬连接”的重压，精神状态都很绷

场景 1：观测点入口

内容要点：
- 进入公交调度中心前，先写一小段停顿。
- 所有人都会盯着那块旧标牌，不太敢继续迈步。
- 陆沉的第一反应不该是激动，而是更冷静：
  - 既然这里是 `观测点`
  - 就说明这地方不是单纯给人逃生用的
  - 它更像某种记录和分流的中转处
- 林晚会先注意到一点现实层面的异常：
  - 这里不像被废弃很久
  - 地上没有积太厚的灰
  - 某些门像刚被人开过
- 周岚则会很明显地不安，觉得这种“像有人还在上班”的地方更瘆人。

这一场的作用：
- 把第20章的“观测点”概念落到现实场景里
- 先让众人意识到，这里不是普通公交站

场景 2：过于正常的调度广场

内容要点：
- 众人真正走入调度广场后，最先感受到的不是恐怖，而是整齐。
- 可以用这些元素立气质：
  - 电子发车牌还在跳字
  - 站牌时刻表是完整的
  - 调度室玻璃窗擦得很干净
  - 司机休息室门半掩着
  - 不远处还停着几辆外观看起来完好的夜班公交
- 但所有“正常”里都有一处不对：
  - 发车时间全是过去的
  - 线路编号对不上现实地图
  - 站名有的缺字，有的用旧称
  - 电子牌显示下一班车即将到站，但站台空无一人
- 陆沉的规则视界在这里会有一种新的表现：
  - 不是密密麻麻的死法提示
  - 而是不同线路上有不同颜色深浅的血光

这一场的作用：
- 把公交副本和地铁彻底拉开
- 第一次让“线路本身有危险等级”具象化

场景 3：欢迎语落地

内容要点：
- 众人刚分散去看不同站牌和电子牌时，调度广场广播突然响起。
- 广播内容不要太长，必须标准、机械，却带一点不对劲。
- 最适合的开场句就是：
  - `欢迎回家。`
- 这四个字出来时，所有人都会下意识停住。
- 接着广播再补一句很短的提示，像是在宣读一套乘车须知的开头：
  - `请乘坐正确线路。`
- 陆沉会明显感觉到，这句欢迎语不像在对所有人说。
- 它更像是：
  - 对编号对象说
  - 对某个已经被记录过的人说
- 最好再给一个更私人、但不说透的细节：
  - 其他人只听见广播
  - 陆沉却像听见尾音在自己耳边又近了一寸
- 林晚会注意到陆沉脸色不对，第一时间去看他，而不是先看别处。

这一场的作用：
- 把 `欢迎回家` 从单纯氛围句升级成带指向性的规则话语
- 强化陆沉和观测系统之间的私人关联感

场景 4：第一组公交规则

内容要点：
- 广播结束后，电子发车牌或LED屏亮起第一组完整规则。
- 规则不要多，保持 3 到 4 条最利落：
  - 不要乘坐没有线路号的车
  - 发车前，不要换座位
  - 如果同一辆车第二次驶回站台，不要再上
  - 听见 `欢迎回家` 后，不要立刻看向窗外
- 众人会本能去理解这些规则，但越看越不安。
- 因为这些规则和地铁不同：
  - 地铁规则是“你在站台和车厢里别做什么”
  - 公交规则更像“系统已经替你安排了一条路，你别试图乱改”
- 可以安排一个普通幸存者脱口而出：
  - 至少规则都写出来了
  - 比医院里那种闷死人的地方强
- 这句台词很重要，它代表普通人会被“规则明示”安慰到。
- 陆沉则会更警惕，因为规则越明确，说明误入的代价越固定。

这一场的作用：
- 正式立住公交副本第一轮玩法
- 让读者一眼看出“公交是路线筛选型副本”

场景 5：队伍的第一轮误判

内容要点：
- 规则亮出后，队伍内部出现一个短暂松口气的瞬间。
- 有人会觉得：
  - 既然这里只要按规则上车就行
  - 说不定比地铁还好应付
- 周岚也会倾向先找个能坐下的地方缓一缓。
- 老李则会提出最现实的问题：
  - 这里总不可能一直待着
  - 早晚还是得上车
- 陆沉不会急着下命令，而是会先观察：
  - 哪些车没有线路号
  - 哪些车明明熄火却在滴水
  - 哪个站台附近的血光最重
- 这里最好顺手埋下一个后续有效的判断：
  - `13路` 不是最红的
  - 但它最“稳”
  - 这说明它不像随机异常，更像专门在等什么人
- 他的判断要开始从“看局部死法”升级为“看整体路线风险”。

这一场的作用：
- 保留普通人的真实反应
- 让主角的能力升级在行动里体现出来

场景 6：13路亮灯

内容要点：
- 众人还在犹豫时，某个停在最边缘站台的老旧公交忽然发出轻响。
- 不是发动机轰鸣，而是更轻的电流接通声。
- 下一秒，车头线路牌自己亮起。
- 上面慢慢显出：
  - `13`
- 这时候不要立刻让人上车，重点是做那一下“所有人都看见了”的冲击。
- 车灯一亮，调度广场其他电子牌可以短暂闪烁一下，像整套系统都跟着同步了。
- 队伍里的第一反应一定会是：
  - 它在叫陆沉
  - 这辆车是不是就是给他准备的
- 陆沉不会马上靠近，反而会本能后退半步。
- 这一退很关键：它说明主角也意识到，`13路` 不是“答案”，而更像一条会主动认人的线路。

章尾钩子：
- 一辆车头写着 `13` 的老旧公交，在无人靠近的情况下，自己亮起了车灯。

本章节奏提醒：
- 第21章重点是“气质切换”和“规则落地”，不是立刻打完整验证
- 公交调度中心必须和前两个副本明显不同，核心词是：`正常、整齐、被安排`
- 章尾要收在 `13路亮灯`，让读者自然追下一章

本章伏笔：
- `欢迎回家` 不只是场景广播，更像对特定对象说的话
- 不同线路本身带有不同危险等级
- 调度中心像某种分流和观测前厅
- `13路` 很可能是给陆沉准备的专属筛选线

本章读者爽点：
- 新副本有明显新鲜感
- 公交机制和地铁彻底拉开
- 章尾 `13路亮灯` 有很强的私人点名感
