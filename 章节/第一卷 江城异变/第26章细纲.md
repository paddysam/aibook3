# 第 26 章细纲 主网外沿

本章定位：
第二十六章是 `公交调度中心阶段性退潮` 和 `城市路线分歧正式抬头` 的过渡章。它不能写成简单的“查完了下一站去哪”，而是要完成两件事：第一，陆沉第一次真正区分 `主网核心 / 外沿连接点 / 假亮点`；第二，幸存者队伍第一次在城市层面出现真实路线分歧。

本章目标：
- 让公交调度中心阶段性失去继续给答案的价值
- 让陆沉第一次看清三类点：主网核心、外沿连接点、假亮点
- 让周岚、老李、林晚分别站到不同的“路线态度”上
- 让“队伍分歧”第一次明牌，但先不要彻底撕开
- 章尾抛出 `真正的入口，不止一个`

本章核心冲突：
- 主冲突：现在到底该去最近的、最亮的，还是最像答案的地方
- 次冲突：主角能力升级了，但幸存者的承受能力并没有同步升级
- 恐怖来源：城市图上的动态亮点、突然失效的节点、调度中心外沿的异动

本章回报类型：
- 本章回报重点是：`认知升级 + 现实阻力正式成形`

承接开头：
- 紧接第25章章尾
- 队伍刚看见：
  - `13号后，系统重启。`
- 老张已被错误线路预定，观测系统的压迫感达到目前最高点

场景 1：调度中心不再继续给答案

内容要点：
- 第25章结尾已经把系统历史抬起来了，第26章开头不能继续翻同一本记录。
- 最合适的处理是：
  - 陆沉还想继续翻调度台
  - 但那些记录开始变空、变旧、变成没价值的壳
  - 或者后续页全部只剩模糊墨迹
- 这会形成一个很重要的判断：
  - 调度中心不是终点
  - 它只是某个阶段的入口或分流前厅
- 这一下要让读者感到：这里不会再给关键答案了，继续停留只会被系统反过来处理

这一场的作用：
- 让公交调度中心自然退潮
- 为离开当前节点建立合理性

场景 2：规则视界的新分层

内容要点：
- 陆沉离开调度台后，再次把规则视界推向整片城市。
- 这一次，他看见的不再只是“有亮点”。
- 他开始区分三类东西：
  - `主网核心`：血光深、稳、像有固定结构
  - `外沿连接点`：比核心弱，但和核心之间像有线相连
  - `假亮点 / 诱饵点`：亮得突然、亮得假，像专门引人过去
- 这个认知不要讲得太系统化，要写得像他在不断试着给这些异常分类。
- 最重要的是：这不是全知，而是第一次分层。

这一场的作用：
- 真正推进主角能力的形态升级
- 让第27章“三个亮点”更有基础

场景 3：现实阻力分化

内容要点：
- 这章必须把队伍意见分化写出来，而且每个人的理由要不同。
- 但要注意，这里是 `分歧抬头`，不是立刻吵到队伍裂开。
- 周岚：
  - 偏向最近、最像安全撤离点的路线
  - 她更关心“能不能先躲一会儿”
- 老李：
  - 更直接，他会觉得继续往里走就是找死
  - 他会希望选一条“至少像现实生活区”的路线
- 林晚：
  - 站在陆沉这边，但不等于她完全不怕
  - 她更像是认可“必须继续查”，而不是认可某一条具体路线
- 老张：
  - 因为第25章已经被“错误线路预定”，状态会明显不对
  - 他不一定主动发言，但他的存在本身会逼得所有人更怕“再选错一次”
- 这些态度都很真实，不能再写成所有人整齐划一地听主角。

这一场的作用：
- 保住群像真实感
- 让后面的路线选择更有压力

场景 4：一个短暂的节点异变

内容要点：
- 为了不让这一章全是说话和判断，必须加一个小的、但很值的现场变化。
- 比如：
  - 某个原本亮着的点突然熄掉
  - 或调度中心外沿某一整片街灯同时暗一下
  - 或某个刚刚还在“主网外沿”的弱光点，突然变成深红
- 这不是为了吓人本身，而是让陆沉和队伍意识到：
  - 城市节点不是静态地图
  - 它们会变化
  - 他们不能磨太久
- 最好再补一个很值钱的小发现：
  - 现实里的变化已经发生了
  - 但规则视界或调度残留记录里，还短暂保留着半拍之前的状态
- 这会非常关键，因为它是后面“系统更新不是绝对同步”的第一次前置证据。

这一场的作用：
- 把“亮点是活的”先试探性抛出来
- 给路线争执施加倒计时压力

场景 5：真正的入口，不止一个

内容要点：
- 在前面几轮判断和阻力之后，陆沉不会直接说“去这里”，而是先确认一个更高一层的结论：
  - 他们之前以为只有一个入口
  - 现在看，不止一个
- 他会在城市图上标出三处点。
- 注意，这里不要立刻讲三处点具体代表什么，只要让读者知道：
  - 主角现在已经不是“找一个方向跑”
  - 而是在“比较三个可能的入口”

章尾钩子：
- 陆沉盯着城市图上那三处不同强弱的血点，低声说：
  - `真正的入口，不止一个。`

本章节奏提醒：
- 第26章不能写成“继续翻资料”
- 必须让“路线问题”第一次真正落地，但先停在“明牌分歧”
- 节点异变要小，但必须有，让这一章不空

本章伏笔：
- 三处点在第27章会被具体化
- 假亮点和真入口的差异，将成为主角后续反制系统的基础
- 系统记录和现实变化之间，可能存在极短的更新滞后
- 队伍里有人已经快承受不住下一轮选择

本章读者爽点：
- 主角能力第一次从“看点”升级到“分层判断点”
- 队伍终于不再只是跟着走，而是出现了现实阻力
- 章尾三点并列，很适合拉下一章
