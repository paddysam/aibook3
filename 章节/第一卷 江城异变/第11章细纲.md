# 第 11 章细纲 地铁站入口

本章定位：
第十一章必须把第十章章尾的电子屏钩子接实，让故事从“医院阶段性脱身”顺滑切进“城市公共空间规则副本”。这一章的重点不是立刻死人，而是建立一种新的怪谈秩序：医院是封闭压迫，地铁是公开、冰冷、像正常运行却处处不正常。

本章目标：
- 接住 `下一站，请遵守规则`
- 让幸存者第一次面对“医院外也必须主动进副本”的现实
- 建立地铁站的空间气质和医院不同
- 让队伍对是否进入地铁站发生分歧
- 让陆沉第一次用规则视界判断“规则节点”而不是只看死法
- 在章尾正式抛出地铁站第一组广播钩子

本章核心冲突：
- 主冲突：进不进地铁站
- 次冲突：幸存者刚出医院，不愿再主动踏进新规则场
- 恐怖来源：空站、电子屏、广播、正常设施仍在运行但无人使用

本章回报类型：
- 本章不是爽点章
- 回报重点是：`新副本正式开启，主角第一次主动判断规则节点`

承接开头：
- 紧接第十章章尾
- 地铁站电子屏刚亮起血字：
  - `下一站，请遵守规则。`
- 众人刚从医院出来，处于“想逃离一切”的本能状态

场景 1：医院门口的停顿

内容要点：
- 电子屏亮起后，所有人都被那行字钉在原地。
- 幸存者第一反应不是靠近，而是本能后退。
- 一个普通家属可以先开口：
  - 医院好不容易出来了
  - 为什么还要去地铁站
- 周岚会明显偏保守，她更想找个能锁门的地方躲起来，而不是继续往规则中心靠。
- 林晚会比其他人更快意识到问题：
  - 不是他们想不去就能不去
  - 既然地铁站主动亮灯，就说明那地方迟早会和他们发生关系

这一场的作用：
- 把“逃出医院后还要继续进副本”的心理阻力写出来
- 让人物分歧真实成立

场景 2：陆沉第一次看“节点”

内容要点：
- 陆沉用规则视界去看医院外的世界。
- 和医院里不同，地铁站没有密密麻麻的具体死亡提示，而是整体血光更集中、更稳定。
- 他第一次对“规则节点”有了更明确的感知：
  - 医院外其他建筑只是零散残字
  - 地铁站入口像一个被主动点亮的规则核心
- 这个判断不要写得像最终术语定义，而是主角自己的朴素认知：
  - 那地方不是最安全
  - 但一定是最关键

这一场的作用：
- 推进主角能力形态
- 为后面“城市节点”概念打最早地基

场景 3：进站还是绕路

内容要点：
- 队伍内部正式分成两派：
  - 一派想顺着大路先远离医院和地铁
  - 一派认为至少地铁站有遮蔽、有设施、有信息
- 赵虎已死，这一章不要再用强硬刺头冲突，而是用普通人的现实恐惧推动分歧。
- 陆沉不会强行说“必须进”，而是给出更让人绝望的判断：
  - 现在整座城市都不正常
  - 盲走不比进站更安全
  - 至少地铁站已经把规则写在了明面上
- 这句很重要：怪谈最可怕的不是规则，而是没规则。

这一场的作用：
- 让进入地铁站的选择更有合理性
- 避免主角像作者代言人一样硬推剧情

场景 4：进入站厅

内容要点：
- 众人在陆沉带头下接近地铁站入口。
- 站厅的气质必须和医院拉开：
  - 地面很干净
  - 闸机还亮着
  - 自动扶梯在运行
  - 广告灯箱正常亮着，但广告内容停在某个静止画面
  - 没有人，却像刚刚还有人流经过
- 越“正常”，越让人不安。
- 陆沉会本能留意：
  - 站厅里的时间显示
  - 闸机是否开放
  - 屏蔽门状态
  - 站内广播是否还在循环

这一场的作用：
- 用“公共空间正常运行但没人”制造新的恐怖质感
- 给读者副本新鲜感

场景 5：第一组残缺规则

内容要点：
- 进入站厅后，电子屏、站内导视牌、广播里开始出现不完整规则提示。
- 不要一口气给完整规则，只给残句最吊胃口：
  - `……未完全停稳前，请勿……`
  - `……候车时，请站在……`
  - `……站名重复播报三次……`
- 幸存者会很焦躁，因为这比没有规则更难受。
- 林晚可以在这里补一句现实层面的观察：
  - 这些规则像是被故意删掉了一半
  - 好像就是想逼人用命去试

这一场的作用：
- 拉出地铁副本的第一轮玩法
- 把读者推进“下一章必须看规则怎么完整亮相”

场景 6：广播响起

内容要点：
- 就在众人还在盯残缺规则时，站厅广播忽然恢复。
- 广播声音要非常标准、冷静，像真正的地铁女声。
- 她说出的内容不能多，但要狠：
  - `列车即将进站，请遵守候车规则。`
- 这一句一出来，整个站厅的灯光、电子屏、风声都像同步了一下。
- 众人会瞬间意识到：副本开始了。

章尾钩子：
- 最后就收在广播上最合适：
  - `列车即将进站，请遵守候车规则。`

本章节奏提醒：
- 本章重点是“进站”和“气质切换”，不是立刻死人
- 地铁副本的第一感受应该是：比医院更公开、更日常，也更冷
- 不要急着把所有规则讲透，残缺更有追读感

本章伏笔：
- 地铁站是城市规则节点之一
- 规则残句是故意缺失，不是系统错误
- 站厅里某些设施还在“正常运行”，说明怪谈在模仿真实秩序

本章读者爽点：
- 新副本的强新鲜感
- 主角第一次从“看死法”升级到“看节点”
- 章尾广播一句就把副本压实
