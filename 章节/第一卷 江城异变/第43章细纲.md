# 第 43 章细纲 特护3区

本章定位：
第四十三章是医院主线真正回潮的一章，但绝不能写成前10章医院副本回锅。`特护3区` 不是十三层，也不是普通旧病区，它更像这整套系统源头里负责 `接回 / 回床 / 继续观察` 的处理病区。重点不是让读者怀旧“又回医院了”，而是要让读者感受到：医院这条线现在已经从“怪谈爆发点”升级成“归档源头层”。

本章目标：
- 把 `特护3区` 这块医院源头病区真正立住
- 让它和前10章医院、十三层明显区分开
- 给出第一组病区规则
- 安排一次和“空床认人 / 推床通道 / 药车秩序”相关的硬验证
- 让周岚第一次在医院源头病区里发挥现实价值
- 章尾把 `L-13：接回失败，继续观察` 直接打到陆沉脸上

本章核心冲突：
- 主冲突：进入特护3区后，众人到底该不该继续往里走，而一旦往里走，又该怎么避免被病区秩序认领到某张床位上
- 次冲突：医院这条线一回潮，周岚、老李等人会天然觉得“这里更邪”，队伍的退缩感会更强
- 恐怖来源：空床位、床头小屏、推床声、夜灯、药车、白布、被压得极低的旧护理秩序

本章回报类型：
- 本章不是大爽点章
- 回报重点是：`医院源头病区气质正式立住`

承接开头：
- 紧接第42章章尾
- 众人已经被旧病床转运电梯送到：
  - `特护3区`
- 刘成在回床通道里已经被某个床号短暂盯住
- 老李手腕上的 `09` 余波仍能在病区边缘给陆沉一点确认方向

场景 1：这不是十三层

内容要点：
- 开头第一件事，就是把 `特护3区` 和前10章医院区隔开。
- 这地方的恐怖气质不能是停电、走廊追杀、广播点名那种熟悉套路。
- 最适合的是：
  - 走廊狭窄
  - 墙皮发黄
  - 床位整齐得过头
  - 床头小屏偶尔自己跳一下状态
  - 护士站台面擦得很干净
  - 推床通道比正常病区宽一点
- 周岚最适合先认出来：
  - 这不是普通病区
  - 这是医院里处理“最重、最不该外传”的那种地方
- 她会自然提供一个非常值钱的现实信息：
  - 特护3区床位少
  - 但每张床的隔离距离比普通病房大
  - 说明这地方本来就不是给普通病人住的

这一场的作用：
- 把医院源头区立成一种新的“秩序型病区恐怖”
- 让周岚第一次在这条主线上提供真正有用的信息

场景 2：病区规则

内容要点：
- 众人刚沿着走廊往里走，病区一端的床头灯会自己亮一下。
- 与此同时，某块床位小屏或护士站旧播报器亮出第一组规则。
- 规则建议控制在 3 到 4 条：
  - 进入病区后，先记床号，不要叫姓名
  - 听见推床声，不要让路
  - 夜灯亮起前，不要碰空床栏杆
  - 药车经过时，不要看白布下面
- 这几条一定要让读者一眼明白：
  - 这里的秩序不是“治疗”
  - 而是“分配、落位、继续观察”
- 老李会下意识说一句很贴普通人的话：
  - 医院里不让看病人名字，还让先记床号，这地方到底把人当什么

这一场的作用：
- 正式立住病区规则
- 把病区的核心逻辑从“护理”扭成“收容”

场景 3：夜灯没亮前，床会先认人

内容要点：
- 这一章必须给一次病区硬验证，但不能和前面等待区坐椅子重复。
- 最合适的是：
  - 刘成因为灰白手指和回床通道里的残留，走到某张空床旁边时，床头小屏突然亮起
  - 屏幕上先跳一下床号，再立刻变成 `待接回`
- 刘成本人甚至没碰床。
- 但他的呼吸节拍会先乱，肤色开始往病床那块冷区靠拢。
- 最直观的表现可以是：
  - 他裸露在外的手背温度迅速下去
  - 说话时嘴角开始泛白
  - 床垫凹痕像是终于找到了重量，慢慢往下沉半寸
- 这一下会非常值钱。
- 因为它告诉读者：
  - 病区里的床不是空的
  - 而是会主动认“该回来的人”

这一场的作用：
- 完成本章最实在的硬验证
- 把“空床认人”这件事写成和前面完全不同的新恐怖来源

场景 4：推床声不是来接人的，是来落位的

内容要点：
- 验证完床位，还得让病区另一条规则也咬一下人。
- 最适合的是：
  - 众人把刘成从床边硬拖开后
  - 走廊尽头传来推床轮子压地的声音
  - 但视野里暂时什么都没有
- 规则说的是：
  - 听见推床声，不要让路
- 普通人的本能一定是侧身、贴墙、让出通道。
- 周岚这时可以起到作用：
  - 她比别人更熟医院秩序
  - 所以她会本能要让路
  - 也更容易被陆沉一把喝住
- 这一下非常值钱。
- 因为它既能体现周岚的职业惯性，也能和规则形成对冲。
- 最后不用把推床整个亮出来。
- 只要让众人看到：
  - 床轮印自己从地上压过去
  - 或白布边角从拐角掠过
  - 他们让没让路，直接决定那声音有没有从他们身边压过去

这一场的作用：
- 再立一条病区规则不是假的
- 也继续把“医院秩序 = 危险流程”这层感觉压实

场景 5：L-13：接回失败，继续观察

内容要点：
- 章尾必须把这一章和陆沉强绑定。
- 最适合的方式是：
  - 众人被逼到病区最深处
  - 一张一直空着的床位屏，忽然从黑屏中亮起
- 屏幕不能写名字。
- 就写状态。
- 最狠的是：
  - `L-13：接回失败，继续观察`
- 这个词条的好处是：
  - 既扣回第38、40章
  - 也说明特护3区这地方直到现在还在盯这条未闭合档案
- 到这里不要再多解释。
- 让所有人意识到：
  - 他们不是“来到了医院源头”
  - 而是“来到了陆沉曾经没被收完的床位旁边”

章尾钩子：
- 病区最深处那张一直空着的床位屏，突然亮起惨白的字：
  - `L-13：接回失败，继续观察。`

本章节奏提醒：
- 第43章重点是“新病区气质”，不能写成旧医院换皮
- 周岚要开始提供现实价值，但不能突然变成强人
- 空床认人的方式要和等待区座位确认明显区分
- 章尾一定要把 `L-13` 和病区床位直接钉到一起

本章伏笔：
- 特护3区的床位和 `L-13` 存在长期关系
- 病区规则强调床号、床位、回床，说明这里更接近“实际落位层”
- 周岚后面还能继续提供病区布局与护理流程方面的信息

本章读者爽点：
- 医院主线回潮，但玩法完全不是前10章那套
- 病区规则和空床认人的画面很有记忆点
- 章尾 `L-13：接回失败，继续观察` 会直接把追读拉进下一章
