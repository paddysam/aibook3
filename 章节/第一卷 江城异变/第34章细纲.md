# 第 34 章细纲 前一批人

本章定位：
第三十四章的任务，是把“编号对象不止陆沉一个”这条线第一次用场景和现场打击坐实。它不能写成大家进一间仓库、翻几样旧物、拿几条线索就结束，而必须做到：线索一出现，现场立刻咬人。`失物招领` 这四个字本来很日常，但在这里，它不是给活人找回东西的，而像是系统把“前一批人被剥下来的边角”整整齐齐地挂起来了。

本章目标：
- 承接第33章章尾的 `失物招领`
- 把“前编号对象”第一次做成能看见、能害怕的具体痕迹
- 让读者明确知道：`01 / 04 / 09 / 13` 都不是抽象数字，而是曾经真实存在过的人
- 安排一次“旧标签会认人、会往活人身上贴”的现场打击
- 让陆沉判断出：这些旧物不是遗失物，而是归档后留下来的“识别残片”
- 给陆沉拿到一条和医院旧记录、和自己高度相关的硬线索
- 章尾用“陆沉笔迹”把身份压迫再往前推一步

本章核心冲突：
- 主冲突：众人到底该不该碰这间 `失物招领` 里的东西，而陆沉又必须从里面找线索
- 次冲突：普通人会越来越害怕“任何接触都会被套进旧编号”，从而本能排斥继续深入
- 恐怖来源：会自己贴人的旧标签、残留旧站名的喃喃声、被归档者留下的生活边角、和陆沉极像的旧笔迹

本章回报类型：
- 本章回报重点是：`前编号对象痕迹实体化 + 旧物现场咬人`

承接开头：
- 紧接第33章章尾
- 众人已经确认：
  - 这片假站台不是交通空间
  - 而是编号对象的等待区
- 现在，侧门已经自己开了一条缝，门后挂着一排旧标签：
  - `01`
  - `04`
  - `09`
  - `13`

场景 1：这间失物招领不收普通东西

内容要点：
- 众人推开侧门后，最先感受到的不是灰尘，而是整齐。
- 这间屋子不能写成脏乱仓库。
- 更合适的气质是：
  - 货架一排排立着
  - 编号绳整齐垂着
  - 老式登记簿平码在桌角
  - 墙上还有领取流程牌
- 但越看越不对：
  - 货架上不是雨伞、外套、包
  - 而是旧工作证、病历夹边角、公交卡、照片、车票、手写便签
  - 有些东西甚至一眼看不出原本是不是完整物件
- 林晚会先察觉到一点最瘆人的地方：
  - 这些东西不是“被落在这里”
  - 更像是从人身上被整齐拆下来、留下来的
- 陆沉在规则视界里，会看到：
  - 每一排货架对应不同深浅的残红
  - 编号越靠后，血光越稳

这一场的作用：
- 先把 `失物招领` 的空间气质立住
- 把“前编号对象留下边角”的概念落到实景

场景 2：旧标签会认人

内容要点：
- 本章必须尽快给一次现场咬人，不能拖太久。
- 最合适的方式是：
  - 这一下最好落在老李身上，而不是一次性普通路人
  - 因为老李本来就是队伍里最稳定的“普通人视角样本”，让他沾上 `09`，后面压力会更具体
  - 他不是随便站过去就中招，最好让他正好贴在离那格最近的位置，手腕和影子一起压进那格货架边缘
  - 一张本来挂在钩子上的旧标签就顺着那截影子，自己贴上了他的手腕
- 标签上原本褪色的数字，瞬间一点点转红：
  - `09`
- 这一幕一定要安静地发生。
- 因为越安静，越有“系统自动认领”的味道。
- 那个人本人一开始甚至没反应过来。
- 下一秒，他会开始非常轻地重复一个旧站名，像在背自己从没去过的换乘路线。
- 不用让他立刻失控。
- 只要做到：
  - 眼神像被往别的年代拉了一下
  - 嘴里开始冒不属于自己的线路词
  - 手腕上那张标签越贴越紧

这一场的作用：
- 完成本章第一轮现场硬打击
- 让“旧物会认人”这条机制成立

场景 3：识别残片，不是纪念品

内容要点：
- 陆沉不能只看到吓人，他要在这一场读懂机制。
- 他会很快判断：
  - 这张标签不是简单诅咒物
  - 而是某个编号对象被归档后留下的“识别残片”
- 最合适的处理方式是：
  - 不能硬撕
  - 越撕越像在确认认领
- 他要利用环境里的旧领取格位，把标签重新送回对应编号那一列，才勉强让它松动一点。
- 在这个过程中，林晚可以帮他做一个很实用的现实辅助：
  - 她快速核对货架顺序和编号层级
  - 帮陆沉找到 `09` 原本对应的挂位
- 这会让本章不只是“主角一个人灵机一动”。
- 同时也把读者认知往前推一步：
  - 这些东西不是前人留下来的纪念品
  - 是系统还拿来继续认人的残片
- 这一轮不要在本章内彻底归零：
  - 老李手腕上最好留下一圈淡红勒痕
  - 耳边或喉咙里还会偶尔残着那个旧站名
  - 这样后面才能把“被编号碰过一次不会立刻清零”继续往下压

这一场的作用：
- 把“旧标签贴人”的惊吓升级成可理解的规则机制
- 让林晚继续承担有效辅助，而不是只围观

场景 4：前一批人真的存在过

内容要点：
- 危险稍微压住后，众人才敢真正看这些东西。
- 这里不能一口气把所有信息讲出来，要走“碎片式坐实”。
- 最合适的是：
  - 摸到某张旧车票，会有一瞬站台残影
  - 翻到某张工作证，能听见一句极短的旧广播
  - 看一张模糊照片，能看见某个人站在同样的分流闸机前
- 重要的是让读者直观看懂：
  - `01 / 04 / 09 / 13` 都曾经是活人
  - 他们不是系统后台生成的编号
- 但这一场不能直接解释他们后来都去了哪。
- 只要留下几个足够瘆人的感觉：
  - 有人被收得很干净，只剩一张编号牌
  - 有人归档失败过，东西比别人多
  - 有人像是卡在“被接进去但又没完全闭合”的状态

这一场的作用：
- 把“前编号对象”从概念变成具体存在
- 给后面的 `04号回声` 提前埋下可信基础

场景 5：队伍第一次真正怕陆沉继续翻下去

内容要点：
- 这一章要顺手补一点人性压力，但先不要提前炸到第38章的程度。
- 最合适的是：
  - 有人看见标签会主动认人，立刻就慌了
  - 觉得再翻下去，只会把更多旧编号招到活人头上
- 老李会明显动摇：
  - 查出来又怎么样
  - 这些东西现在还会往人身上贴
  - 而且他自己已经被 `09` 贴过一次，这种恐惧要比别人更实
- 周岚的反应会更现实：
  - 她不是不想知道真相
  - 是开始害怕“知道得越多，越像已经被选中”
- 这时候陆沉不能停。
- 因为他会在一件和医院旧记录相关的东西上，看到一眼就没法绕开的细节。

这一场的作用：
- 把普通人的真实恐惧保留下来
- 让陆沉继续推进变得更有压力，而不是自然流畅

场景 6：陆沉的笔迹

内容要点：
- 本章最值钱的硬线索，建议放在一张旧标签或旧便签背面。
- 这件东西最好能同时和医院线、编号线都咬上：
  - 比如一张夹在病历夹边角里的旧领取标签
  - 或一张写着某个旧病区简写的登记便签
- 陆沉翻过来后，背面会出现两个很轻的字：
  - `陆沉`
- 这一下还不够，还要再补一刀：
  - 这两个字的笔迹，竟和陆沉现在写字的手势极像
- 这一点非常狠。
- 因为它会让人直接联想到：
  - 这不是“有人认识陆沉”
  - 而像是陆沉自己曾经在这套系统里留下过字
- 但本章不要解释。
- 只收在“这个笔迹不该出现，却偏偏最像他自己”。

章尾钩子：
- 一张旧标签翻过来，背面用极淡的笔写着两个字：
  - `陆沉`
- 而那笔迹，竟和陆沉现在写字的手势极像。

本章节奏提醒：
- 第34章绝不能写成纯线索章，旧物必须先咬人，再给信息
- “前编号对象存在过”这件事要靠残响和边角坐实，不要靠长篇解释
- 本章要把恐怖从“等待区会锁人”升级到“前人留下来的残片还会继续认领活人”
- 章尾的笔迹线很值钱，只负责压迫，不负责讲透

本章伏笔：
- `09` 这种旧标签还能认人，说明归档残片仍有持续效力
- 前编号对象有人收得干净，有人像归档失败过
- 陆沉和这套系统的关系，可能不是“现在才被点名”这么简单
- 笔迹问题会为后面 `接回现实 / 归位失败` 铺一层更早的联系

本章读者爽点：
- `失物招领` 这个日常空间在怪谈里会很有反差新鲜感
- 旧标签自己贴上手腕、数字一点点转红，这种画面很好记
- 章尾“陆沉笔迹”能把身份压迫再狠狠干一刀
